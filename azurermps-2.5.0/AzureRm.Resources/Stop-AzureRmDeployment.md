# Stop-AzureRmDeployment

## 摘要
Cancal 正在執行的部署

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 句法

### StopByDeploymentName (預設) 
```
Stop-AzureRmDeployment [-Name] <String> [-PassThru] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### StopByDeploymentId
```
Stop-AzureRmDeployment -Id <String> [-PassThru] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### StopByInputObject
```
Stop-AzureRmDeployment -InputObject <PSDeployment> [-PassThru] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 說明
**AzureRmDeployment** Cmdlet 會在已開始但尚未完成的訂閱範圍中取消部署。
若要停止部署，部署必須具有不完整的提供狀態（例如 [提供]），而不是已完成的狀態（例如 [已預配] 或 [未成功]）。

若要在訂閱範圍建立部署，請使用 New-AzureRmDeployment Cmdlet。

這個 Cmdlet 只會停止一個正在執行的部署。 使用 *Name* 參數停止特定的部署。

## 示例

### 範例1
```
PS C:\>Stop-AzureRmDeployment -Name "deployment01"
```

這個命令會在目前的訂閱範圍中取消正在執行的部署 "deployment01"。

### 範例2
```
PS C:\>Get-AzureRmDeployment -Name "deployment01" | Stop-AzureRmDeployment
```

這個命令會在目前的訂閱範圍中取得部署 "deployment01"，並加以取消。
