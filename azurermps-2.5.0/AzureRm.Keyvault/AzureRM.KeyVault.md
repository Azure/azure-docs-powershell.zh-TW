# AzureRM KeyVault Module
## 說明
本主題顯示 Azure 金鑰 Vault Cmdlet 的說明主題。

## AzureRM KeyVault Cmdlet
### [附加 AzureKeyVaultCertificate](Add-AzureKeyVaultCertificate.md)
將憑證新增到金鑰保存庫。

### [附加 AzureKeyVaultCertificateContact](Add-AzureKeyVaultCertificateContact.md)
新增憑證通知的連絡人。

### [附加 AzureKeyVaultKey](Add-AzureKeyVaultKey.md)
在金鑰保存庫中建立金鑰或將金鑰匯入到金鑰保存庫中。

### [附加 AzureKeyVaultManagedStorageAccount](Add-AzureKeyVaultManagedStorageAccount.md)
將現有的 Azure 儲存空間帳戶新增到指定的 [金鑰保存庫]，以便由主要 Vault 服務來管理其金鑰。

### [備份-AzureKeyVaultKey](Backup-AzureKeyVaultKey.md)
備份金鑰保存庫中的金鑰。

### [備份-AzureKeyVaultSecret](Backup-AzureKeyVaultSecret.md)
在金鑰保存庫中備份密碼。

### [AzureKeyVaultCertificate](Get-AzureKeyVaultCertificate.md)
從金鑰保存庫取得憑證。

### [AzureKeyVaultCertificateContact](Get-AzureKeyVaultCertificateContact.md)
取得針對金鑰保存庫的憑證通知所註冊的連絡人。

### [AzureKeyVaultCertificateIssuer](Get-AzureKeyVaultCertificateIssuer.md)
取得金鑰保存庫的憑證頒發者。

### [AzureKeyVaultCertificateOperation](Get-AzureKeyVaultCertificateOperation.md)
取得憑證操作的狀態。

### [AzureKeyVaultCertificatePolicy](Get-AzureKeyVaultCertificatePolicy.md)
在金鑰保存庫中取得憑證的原則。

### [AzureKeyVaultKey](Get-AzureKeyVaultKey.md)
取得金鑰保存庫金鑰。

### [AzureKeyVaultManagedStorageAccount](Get-AzureKeyVaultManagedStorageAccount.md)
取得主要電子倉庫管理的 Azure 儲存空間帳戶。

### [AzureKeyVaultManagedStorageSasDefinition](Get-AzureKeyVaultManagedStorageSasDefinition.md)
取得主要電子倉庫管理的儲存 SAS 定義。

### [AzureKeyVaultSecret](Get-AzureKeyVaultSecret.md)
取得金鑰保存庫中的密碼。

### [AzureRmKeyVault](Get-AzureRmKeyVault.md)
取得金鑰電子倉庫。

### [匯入-AzureKeyVaultCertificate](Import-AzureKeyVaultCertificate.md)
將憑證匯入到金鑰保存庫。

### [新-AzureKeyVaultCertificateAdministratorDetails](New-AzureKeyVaultCertificateAdministratorDetails.md)
建立記憶體內部的憑證系統管理員詳細資料物件。

### [新-AzureKeyVaultCertificateOrganizationDetails](New-AzureKeyVaultCertificateOrganizationDetails.md)
建立記憶體內部的憑證組織詳細資料物件。

### [新-AzureKeyVaultCertificatePolicy](New-AzureKeyVaultCertificatePolicy.md)
建立記憶體內部的憑證原則物件。

### [新-AzureRmKeyVault](New-AzureRmKeyVault.md)
建立金鑰電子倉庫。

### [移除-AzureKeyVaultCertificate](Remove-AzureKeyVaultCertificate.md)
移除金鑰保存庫的憑證。

### [移除-AzureKeyVaultCertificateContact](Remove-AzureKeyVaultCertificateContact.md)
從金鑰保存庫刪除已登錄到證書通知的連絡人。

### [移除-AzureKeyVaultCertificateIssuer](Remove-AzureKeyVaultCertificateIssuer.md)
從金鑰保存庫刪除憑證簽發者。

### [移除-AzureKeyVaultCertificateOperation](Remove-AzureKeyVaultCertificateOperation.md)
從金鑰保存庫刪除憑證操作。

### [移除-AzureKeyVaultKey](Remove-AzureKeyVaultKey.md)
刪除主要電子倉庫中的金鑰。

### [移除-AzureKeyVaultManagedStorageAccount](Remove-AzureKeyVaultManagedStorageAccount.md)
移除主要電子倉庫管理的 Azure 儲存空間帳戶和所有相關的 SAS 定義。

### [移除-AzureKeyVaultManagedStorageSasDefinition](Remove-AzureKeyVaultManagedStorageSasDefinition.md)
移除金鑰保存庫管理的 Azure 儲存體 SAS 定義。

### [移除-AzureKeyVaultSecret](Remove-AzureKeyVaultSecret.md)
刪除金鑰保存庫中的密碼。

### [移除-AzureRmKeyVault](Remove-AzureRmKeyVault.md)
刪除金鑰電子倉庫。

### [移除-AzureRmKeyVaultAccessPolicy](Remove-AzureRmKeyVaultAccessPolicy.md)
從金鑰保存庫移除使用者或應用程式的擁有權限。

### [Restore-AzureKeyVaultKey](Restore-AzureKeyVaultKey.md)
從備份金鑰在金鑰保存庫中建立金鑰。

### [Restore-AzureKeyVaultSecret](Restore-AzureKeyVaultSecret.md)
從備份的機密在金鑰保存庫中建立秘密。

### [Set-AzureKeyVaultCertificateAttribute](Set-AzureKeyVaultCertificateAttribute.md)
修改憑證的可編輯屬性。

### [Set-AzureKeyVaultCertificateIssuer](Set-AzureKeyVaultCertificateIssuer.md)
在金鑰保存庫中設定憑證簽發者。

### [Set-AzureKeyVaultCertificatePolicy](Set-AzureKeyVaultCertificatePolicy.md)
在金鑰保存庫中建立或更新憑證的原則。

### [Set-AzureKeyVaultKeyAttribute](Set-AzureKeyVaultKeyAttribute.md)
更新金鑰保存庫中金鑰的屬性。

### [Set-AzureKeyVaultManagedStorageSasDefinition](Set-AzureKeyVaultManagedStorageSasDefinition.md)
針對特定的主要電子倉庫管理的 Azure 儲存空間帳戶，使用金鑰保存庫 (SAS) 定義來設定共用存取簽章。

### [Set-AzureKeyVaultSecret](Set-AzureKeyVaultSecret.md)
在金鑰保存庫中建立或更新密碼。

### [Set-AzureKeyVaultSecretAttribute](Set-AzureKeyVaultSecretAttribute.md)
更新金鑰保存庫中密碼的屬性。

### [Set-AzureRmKeyVaultAccessPolicy](Set-AzureRmKeyVaultAccessPolicy.md)
授與修改使用者、應用程式或安全性群組的現有許可權，以使用金鑰保存庫執行作業。

### [停止 AzureKeyVaultCertificateOperation](Stop-AzureKeyVaultCertificateOperation.md)
在金鑰保存庫中取消憑證操作。

### [復原-AzureKeyVaultCertificateRemoval](Undo-AzureKeyVaultCertificateRemoval.md)
將 [金鑰 vault] 中已刪除的憑證恢復為 [作用中] 狀態。

### [復原-AzureKeyVaultKeyRemoval](Undo-AzureKeyVaultKeyRemoval.md)
將 [金鑰 vault] 中的已刪除金鑰復原為 [作用中] 狀態。

### [復原-AzureKeyVaultSecretRemoval](Undo-AzureKeyVaultSecretRemoval.md)
將 [金鑰 vault] 中已刪除的機密恢復為 [作用中] 狀態。

### [復原-AzureRmKeyVaultRemoval](Undo-AzureRmKeyVaultRemoval.md)
將已刪除的金鑰電子倉庫恢復成作用中狀態。

### [更新-AzureKeyVaultManagedStorageAccount](Update-AzureKeyVaultManagedStorageAccount.md)
更新主要電子倉庫管理的 Azure 儲存空間帳戶的可編輯屬性。

### [更新-AzureKeyVaultManagedStorageAccountKey](Update-AzureKeyVaultManagedStorageAccountKey.md)
重新生成主要電子倉庫管理的 Azure 儲存空間帳戶的指定金鑰。
