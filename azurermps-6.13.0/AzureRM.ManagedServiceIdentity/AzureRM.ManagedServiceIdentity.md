# AzureRM ManagedServiceIdentity Module
## 說明
本主題顯示 Azure Managed 服務身分識別 Cmdlet 的說明主題。

## AzureRM ManagedServiceIdentity Cmdlet
### [AzureRmUserAssignedIdentity](Get-AzureRmUserAssignedIdentity.md)
取得使用者指派的身分識別/身分識別。

### [新-AzureRmUserAssignedIdentity](New-AzureRmUserAssignedIdentity.md)
建立指派身分識別的新使用者，或更新現有的指派身分識別身分識別的使用者。

### [移除-AzureRmUserAssignedIdentity](Remove-AzureRmUserAssignedIdentity.md)
移除指派身分識別的使用者。
