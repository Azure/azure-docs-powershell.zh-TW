# Remove-AzureRmDiagnosticSetting

## 摘要
移除 a 資源的診斷設定。

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 句法

```
Remove-AzureRmDiagnosticSetting -ResourceId <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 說明
**AzureRmDiagnosticSetting** Cmdlet 會移除特定資源的診斷設定。
這個 Cmdlet 會實現 ShouldProcess 模式，亦即，在實際建立、修改或移除資源之前，它可能會要求使用者進行確認。

## 示例

### 範例1：移除資源的預設診斷設定 (服務) 
```
PS C:\>Remove-AzureRmDiagnosticSetting -ResourceId "Resource01"
```

這個命令會移除稱為 Resource01 之資源的預設診斷設定 (服務) 。

### 範例2：移除資源指定名稱所識別的預設診斷設定
```
PS C:\>Remove-AzureRmDiagnosticSetting -ResourceId "Resource01" -Name myDiagSetting
```

這個命令會針對稱為 Resource01 的資源，移除稱為 myDiagSetting 的診斷設定。
