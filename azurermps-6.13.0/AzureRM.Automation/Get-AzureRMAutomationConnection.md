# Get-AzureRmAutomationConnection

## 摘要
取得自動化連線。

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 句法

### ByAll (預設) 
```
Get-AzureRmAutomationConnection [-ResourceGroupName] <String> [-AutomationAccountName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByConnectionName
```
Get-AzureRmAutomationConnection [-Name] <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ByConnectionTypeName
```
Get-AzureRmAutomationConnection [-ConnectionTypeName] <String> [-ResourceGroupName] <String>
 [-AutomationAccountName] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 說明
**AzureRmAutomationConnection** Cmdlet 會取得一或多個 Azure 自動化連線。
根據預設，這個 Cmdlet 會檢索所有連線。
指定連線的名稱以取得特定連線。
指定連線類型名稱，以取得特定類型的所有連線。

## 示例

### 範例1：取得所有連線
```
PS C:\>Get-AzureRmAutomationConnection -ResourceGroupName "ResourceGroup01" -AutomationAccountName "Contoso17"
```

這個命令會針對自動化帳戶中名為 Contoso17 的所有連線取得中繼資料。

### 範例2：取得某個類型的所有連接
```
PS C:\>Get-AzureRmAutomationConnection -ResourceGroupName "ResourceGroup01" -AutomationAccountName "Contoso17" -ConnectionTypeName "SqlServer"
```

這個命令會針對自動化帳戶（名為 Contoso17）中的連線取得中繼資料。
這個命令會取得類型 SqlServer 的連線。

### 範例3：取得連線
```
PS C:\>Get-AzureRmAutomationConnection -ResourceGroupName "ResourceGroup01" -AutomationAccountName "Contoso17" -Name "ContosoConnection"
```

這個命令會取得名為 ContosoConnection 之連接的中繼資料。
