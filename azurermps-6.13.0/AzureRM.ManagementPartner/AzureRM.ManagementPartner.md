# AzureRM ManagementPartner Module
## 說明
Azure 資源管理員的管理合作夥伴 Cmdlet。

## AzureRM ManagementPartner Cmdlet
### [AzureRmManagementPartner](Get-AzureRmManagementPartner.md)
取得 Microsoft 合作夥伴網路 (MPN 目前已驗證的使用者或服務主體) 識別碼。 

### [新-AzureRmManagementPartner](New-AzureRmManagementPartner.md)
將 Microsoft Partner Network (MPN) 識別碼與目前已驗證身份的使用者或服務主體相關聯。

### [移除-AzureRmManagementPartner](Remove-AzureRmManagementPartner.md)
刪除 [Microsoft 合作夥伴網路] (MPN 目前已驗證驗證的使用者或服務主體的) ID。

### [更新-AzureRmManagementPartner](Update-AzureRmManagementPartner.md)
更新 Microsoft 合作夥伴網路 (MPN 目前已驗證的使用者或服務主體) 識別碼。
