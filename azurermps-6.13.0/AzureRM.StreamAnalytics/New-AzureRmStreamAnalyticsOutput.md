# New-AzureRmStreamAnalyticsOutput

## 摘要
建立或更新資料流程分析作業的輸出。

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 句法

```
New-AzureRmStreamAnalyticsOutput [-JobName] <String> [[-Name] <String>] [-File] <String> [-Force]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 說明
**AzureRmStreamAnalyticsOutput** Cmdlet 會在串流分析作業中建立輸出，或更新現有的輸出。
您可以在中指定輸出的名稱。JSON 檔案或命令列上。
如果兩者都指定，則命令列上的名稱必須與檔案中的名稱相符。
如果您指定的輸出已存在，但未指定 *Force* 參數，則 Cmdlet 會詢問是否要取代現有的輸出。
如果您指定 *Force* 參數並指定現有的輸出名稱，則會在未確認的情況下取代輸出。

## 示例

### 範例1：將輸出新增至作業
```
PS C:\>New-AzureRmStreamAnalyticsOutput -ResourceGroupName "StreamAnalytics-Default-West-US" -File "C:\Output.json" -JobName "StreamingJob" -Name "Output"
```

這個命令會在名為 StreamingJob 的作業中建立稱為輸出的新輸出。
如果已定義具有此名稱的現有輸出，該 Cmdlet 會詢問是否要取代它。

### 範例2：取代作業輸出定義
```
PS C:\>New-AzureRmStreamAnalyticsOutput -ResourceGroupName "StreamAnalytics-Default-West-US" -File "C:\Output.json" -JobName "StreamingJob" -Name "Output" -Force
```

這個命令會取代作業中名為 StreamingJob 的輸出定義，無需確認。
