# Get-AzureRmNetworkInterfaceIpConfig

## 摘要
取得網路介面的網路介面 IP 配置。

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 句法

```
Get-AzureRmNetworkInterfaceIpConfig [-Name <String>] -NetworkInterface <PSNetworkInterface>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 說明
**AzureRmNetworkInterfaceIPConfig** Cmdlet 會從 Azure 網路介面取得網路介面 IP 配置。

## 示例

### 1：取得網路介面的 IP 設定
```
$nic1 = Get-AzureRmNetworkInterface -Name mynic -ResourceGroupName $myrg
Get-AzureRmNetworkInterfaceIpConfig -Name ipconfig1 -NetworkInterface $nic1
```

第一個命令會取得名為 mynic 的現有網路介面，並將它儲存在 $nic 1 的變數中。 第二個命令會取得此網路介面的名為 ipconfig1 的 IP 配置。
