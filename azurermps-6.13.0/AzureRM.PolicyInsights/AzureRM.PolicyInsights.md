# AzureRM PolicyInsights Module
## 說明
本主題顯示 Azure Policy Insights Cmdlet 的說明。

## AzureRM PolicyInsights Cmdlet
### [AzureRmPolicyEvent](Get-AzureRmPolicyEvent.md)
取得建立或更新資源時產生的原則評估事件。

### [AzureRmPolicyRemediation](Get-AzureRmPolicyRemediation.md)
取得原則 remediations。

### [AzureRmPolicyState](Get-AzureRmPolicyState.md)
取得資源的原則合規性狀態。

### [AzureRmPolicyStateSummary](Get-AzureRmPolicyStateSummary.md)
取得資源的最新原則規範狀態摘要。

### [移除-AzureRmPolicyRemediation](Remove-AzureRmPolicyRemediation.md)
刪除原則修正。

### [開始-AzureRmPolicyRemediation](Start-AzureRmPolicyRemediation.md)
針對原則指派建立及啟動原則修正。

### [停止 AzureRmPolicyRemediation](Stop-AzureRmPolicyRemediation.md)
取消進行中的原則修正。
