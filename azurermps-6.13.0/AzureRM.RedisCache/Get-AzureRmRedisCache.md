# Get-AzureRmRedisCache

## 摘要
取得 Redis 快取。

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 句法

```
Get-AzureRmRedisCache [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## 說明
AzureRmRedisCache Cmdlet 會取得指定的 Azure Redis 快 **取** 。
如果您沒有指定參數，此操作會取得目前訂閱的每個 Redis 快取。

## 示例

### 範例1：依名稱取得 Redis 快取
```

        Id                 : /subscriptions/a559b6fd-3a84-40bb-a450-b0db5ed37dfe/resourceGroups/myGroup/providers/Microsoft.Cache/Redis/myexists

        Name               : myexists
        Type               : Microsoft.Cache/Redis
        HostName           : myexists.redis.cache.windows.net
        Port               : 6379
        ProvisioningState  : succeeded
        SslPort            : 6380
        RedisConfiguration : {}
        EnableNonSslPort   : False
        RedisVersion       : 2.8
        Size               : 1GB
        Sku                : Basic
        Tag                : {}
        Zone               : []
```

這個命令會取得名為 myexists 的 Redis 快取。
