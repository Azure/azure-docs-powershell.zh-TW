# AzureRM SignalR Module
## 說明
與 Azure SignalR service 互動的 Cmdlet。

## AzureRM SignalR Cmdlet
### [AzureRmSignalR](Get-AzureRmSignalR.md)
取得資源群組或訂閱中的特定 SignalR 服務或所有 SignalR 服務。

### [AzureRmSignalRKey](Get-AzureRmSignalRKey.md)
取得 SignalR 服務的便捷鍵。

### [新-AzureRmSignalR](New-AzureRmSignalR.md)
建立 SignalR 服務。

### [新-AzureRmSignalRKey](New-AzureRmSignalRKey.md)
重新產生 SignalR 服務的便捷鍵。

### [移除-AzureRmSignalR](Remove-AzureRmSignalR.md)
移除 SignalR 服務。
