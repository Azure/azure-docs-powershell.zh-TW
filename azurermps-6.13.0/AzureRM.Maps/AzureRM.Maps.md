# AzureRM 圖模組
## 說明
本主題顯示 Azure Maps Cmdlet 的說明。

## AzureRM Cmdlet
### [AzureRmMapsAccount](Get-AzureRmMapsAccount.md)
取得帳戶。

### [AzureRmMapsAccountKey](Get-AzureRmMapsAccountKey.md)
取得帳戶的 API 金鑰。
這些金鑰是在對 Azure 對應進行後續呼叫時所使用的驗證機制。

### [新-AzureRmMapsAccount](New-AzureRmMapsAccount.md)
建立 Azure 地圖帳戶。

### [新-AzureRmMapsAccountKey](New-AzureRmMapsAccountKey.md)
重新生成帳戶金鑰。

### [移除-AzureRmMapsAccount](Remove-AzureRmMapsAccount.md)
刪除 Azure 地圖帳戶。
