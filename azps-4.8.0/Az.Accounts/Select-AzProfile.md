# Select-AzProfile

## 摘要
針對支援多個服務設定檔的模組，請載入與指定的服務設定檔相對應的 Cmdlet。

## 句法

```
Select-AzProfile -Name <String> [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 說明
針對支援多個服務設定檔的模組，請載入與指定的服務設定檔相對應的 Cmdlet。

## 示例

### 範例1
```powershell
PS C:\> Select-AzProfile hybrid-2019-03
```

從2019年3月載入 AzureStack 設定檔的 Cmdlet
