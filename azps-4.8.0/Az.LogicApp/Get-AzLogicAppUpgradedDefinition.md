# Get-AzLogicAppUpgradedDefinition

## 摘要
取得邏輯 app 的升級定義。

## 句法

```
Get-AzLogicAppUpgradedDefinition -ResourceGroupName <String> -Name <String> -TargetSchemaVersion <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 說明
**AzLogicAppUpgradedDefinition** Cmdlet 會從資源群組取得架構版本與邏輯 app 的升級定義。
這個 Cmdlet 會傳回代表已升級之邏輯 app 之定義的物件。
指定資源群組名稱、邏輯 app 名稱和目標架構版本。
此模組支援動態參數。
若要使用動態參數，請在命令中輸入。
若要探索動態參數的名稱，請在 Cmdlet 名稱後輸入連字號 ( ) ，然後重複按 Tab 鍵以迴圈顯示可用的參數。
如果您省略必要的範本參數，則 Cmdlet 會提示您輸入值。
