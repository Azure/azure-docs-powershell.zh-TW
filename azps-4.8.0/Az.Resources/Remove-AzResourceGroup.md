# Remove-AzResourceGroup

## 摘要
移除 [資源] 群組。

## 句法

### RemoveByResourceGroupName (預設) 
```
Remove-AzResourceGroup [-Name] <String> [-Force] [-AsJob] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveByResourceGroupId
```
Remove-AzResourceGroup -Id <String> [-Force] [-AsJob] [-ApiVersion <String>] [-Pre]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 說明
**AzResourceGroup** Cmdlet 會從目前的訂閱中移除 Azure 資源群組及其資源。
若要刪除資源，但要保留資源群組，請使用 Remove-AzResource Cmdlet。

## 示例

### 範例1：移除資源群組
```
PS C:\>Remove-AzResourceGroup -Name "ContosoRG01"
```

這個命令會從訂閱中移除 ContosoRG01 資源群組。
此 Cmdlet 會提示您進行確認，且不會傳回輸出。

### 範例2：移除資源群組而不進行確認
```
PS C:\>Get-AzResourceGroup -Name "ContosoRG01" | Remove-AzResourceGroup -Force
```

這個命令會使用 Get-AzResourceGroup Cmdlet 來取得資源群組 ContosoRG01，然後透過使用管線運算子將它傳遞給 **Remove-AzResourceGroup** 。 *Force* 參數會取消確認提示。

### 範例3：移除所有資源群組
```
PS C:\>Get-AzResourceGroup | Remove-AzResourceGroup
```

這個命令會使用 **AzResourceGroup** Cmdlet 取得所有資源群組，然後透過使用管線運算子將它們傳遞給 **Remove-AzResourceGroup** 。
