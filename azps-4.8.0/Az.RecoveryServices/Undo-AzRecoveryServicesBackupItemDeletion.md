### <span data-ttu-id="c403d-108">範例1</span><span class="sxs-lookup"><span data-stu-id="c403d-108">Example 1</span></span>
```powershell

PS C:\> $PI = Get-AzRecoveryServicesBackupItem -Container $Cont[0] -WorkloadType AzureVM 
PS C:\> Disable-AzRecoveryServicesBackupProtection -Item $PI[0] -RemoveRecoveryPoints
PS C:\> $PI = Get-AzRecoveryServicesBackupItem -Container $Cont[0] -WorkloadType AzureVM | Where-Object {$_.DeleteState -eq "ToBeDeleted"}
PS C:\> Undo-AzRecoveryServicesBackupItemDeletion -Item $PI[0]
```

<span data-ttu-id="c403d-112">第四個命令會取得處於 softdeleted 狀態的專案。</span><span class="sxs-lookup"><span data-stu-id="c403d-112">The fourth command gets the item which is in a softdeleted state.</span></span>
