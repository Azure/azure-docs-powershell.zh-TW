# New-AzPublicIpTag

## 摘要
建立 IP 標籤。

## 句法

```
New-AzPublicIpTag -IpTagType <String> -Tag <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## 說明
**新的-AzPublicIpTag** Cmdlet 會建立 IP 標記。

## 示例

### 範例1：建立新的 IP 標籤
```powershell
$ipTag = New-AzPublicIpTag -IpTagType $ipTagType -Tag $tag
```

這個命令會使用 Tagtype （例如「FirstPartyUsage」）及標記（例如「/Sql」）來建立新的 IP 標記。 在 publicIpAddress 建立中使用這些特定標籤來進行分配。
