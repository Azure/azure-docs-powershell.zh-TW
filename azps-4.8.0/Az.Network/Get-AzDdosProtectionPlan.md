# Get-AzDdosProtectionPlan

## 摘要
取得 DDoS 保護方案。

## 句法

```
Get-AzDdosProtectionPlan [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 說明
Get-AzDdosProtectionPlan Cmdlet 會取得 DDoS 保護方案。

## 示例
