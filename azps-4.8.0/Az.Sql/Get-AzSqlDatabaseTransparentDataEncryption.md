# Get-AzSqlDatabaseTransparentDataEncryption

## 摘要
取得資料庫的 TDE 狀態。

## 句法

```
Get-AzSqlDatabaseTransparentDataEncryption [-ServerName] <String> [-DatabaseName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 說明
**AzSqlDatabaseTransparentDataEncryption** Cmdlet 會取得 Azure SQL Database 的透明資料加密的狀態 (TDE) 。
如需詳細資訊，請參閱 https://msdn.microsoft.com/library/dn948096 https://msdn.microsoft.com/library/dn948096) Microsoft 開發人員網路文件庫中與 Azure SQL Database (的透明資料加密。
這個 Cmdlet 會取得 TDE 的目前狀態，但加密和解密可能是長時間執行的作業。
若要查看加密掃描進度，請執行 Get-AzSqlDatabaseTransparentDataEncryptionActivity Cmdlet。
Azure 上的 SQL Server Stretch Database 服務也支援此 Cmdlet。

## 示例

### 範例1：取得資料庫的 TDE 狀態
```
PS C:\>Get-AzSqlDatabaseTransparentDataEncryption -ServerName "server01" -ResourceGroupName "resourcegroup01" -DatabaseName "database01"
ResourceGroupName             ServerName                    DatabaseName                                          State
-----------------             ----------                    ------------                                          -----
resourcegroup01               server01                      database01                                            Disabled
```

這個命令會取得名為 server01 之伺服器上名為 Database01 之資料庫的 TDE 狀態。
