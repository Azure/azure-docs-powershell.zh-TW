# Enable-AzSqlServerAdvancedDataSecurity

## 摘要
在伺服器上啟用 [高級資料安全性]。

## 句法

```
Enable-AzSqlServerAdvancedDataSecurity [-DoNotConfigureVulnerabilityAssessment] [-AsJob]
 [-DeploymentName <String>] [-InputObject <AzureSqlServerModel>] -ServerName <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 說明
**Enable-AzSqlServerAdvancedDataSecurity** Cmdlet 可在伺服器上啟用高級資料安全性。 [高級資料安全性] 是一個整合的安全性套件，其中包括您伺服器的資料分類、漏洞評估及高級威脅防護。  (會自動建立新的儲存空間帳戶來儲存漏洞評量。 如果先前已針對此目的建立儲存空間帳戶，則會改為使用它) 

## 示例

### 範例1：啟用伺服器高級資料安全性
```powershell
PS C:\>  Enable-AzSqlServerAdvancedDataSecurity `
            -ResourceGroupName "ResourceGroup01" `
            -ServerName "Server01" 

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : True
```

### 範例2：從伺服器資源啟用伺服器高級資料安全性
```powershell
PS C:\>  Get-AzSqlServer `
           -ResourceGroupName "ResourceGroup01" `
           -ServerName "Server01" `
           | Enable-AzSqlServerAdvancedDataSecurity

ResourceGroupName            : ResourceGroup01
ServerName                   : Server01
IsEnabled                    : True
```

### 範例3

在伺服器上啟用 [高級資料安全性]。  (自動生成) 

```powershell
<!-- Aladdin Generated Example --> 
Enable-AzSqlServerAdvancedDataSecurity -DoNotConfigureVulnerabilityAssessment -ResourceGroupName MyResourceGroup -ServerName s1
```
