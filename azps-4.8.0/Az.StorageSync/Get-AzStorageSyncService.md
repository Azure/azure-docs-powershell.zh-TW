# Get-AzStorageSyncService

## 摘要
這個命令會列出 [訂閱/資源群組] 的指定範圍內的所有儲存同步處理服務。

## 句法

### ParentStringParameterSet (預設) 
```
Get-AzStorageSyncService [[-ResourceGroupName] <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### StringParameterSet
```
Get-AzStorageSyncService [-ResourceGroupName] <String> [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 說明
這個命令會列出 [訂閱/資源群組] 的指定範圍內的所有儲存同步處理服務。 您也可以使用它來列出每個儲存空間同步處理服務的屬性。

## 示例

### 範例1
```powershell
PS C:\> Get-AzStorageSyncService -ResourceGroupName "myResourceGroup"
```

這個命令會列出指定資源群組內的所有儲存同步處理服務資源。 您也可以使用它來列出每個儲存空間同步處理服務的屬性。 指定-StorageSyncServiceName 以傳回特定的專案。
