# Update-AzWebAppTrafficRouting

## 摘要
將路由規則更新到該槽。

## 句法

```
Update-AzWebAppTrafficRouting -ResourceGroupName <String> -WebAppName <String> -RoutingRule <Hashtable>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 說明
**更新-AzWebAppTrafficRouting** Cmdlet 會更新 Azure Web App 槽的路由規則配置。

## 示例

### 範例1更新路由規則，以將15% 的生產 traffice 傳送到 Stg 槽
```powershell
PS C:\>Update-AzWebAppTrafficRouting -ResourceGroupName "Default-Web-WestUS" -WebAppName "ContosoSite" 
- RoutingRule @{AtionHostName='XXXX.azurewebsites.net';ReroutePercentage=15;Name='Stg'}
```

這個命令會更新路由規則，以將15% 的生產流量轉移至 Stg 槽。

### 範例2更新路由規則，將生產 traffice 轉換成以遞增方式從50% 轉移到90% 的 Stg 槽範圍。
```powershell
PS C:\>Update-AzWebAppTrafficRouting -ResourceGroupName "Default-Web-WestUS" -WebAppName "ContosoSite" 
-RoutingRule @{ActionHostName='XXXX.azurewebsites.net';ReroutePercentage=50;ChangeIntervalInMinutes=1;
MinReroutePercentage=50;MaxReroutePercentage=90;Name='Stg';ChangeStep=10}
```

這個命令會更新路由規則，將生產 traffice 從50% 轉移到90% （以遞增方式）。
