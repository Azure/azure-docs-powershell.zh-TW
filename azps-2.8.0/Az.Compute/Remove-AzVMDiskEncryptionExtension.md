# Remove-AzVMDiskEncryptionExtension

## 摘要
從虛擬機器移除磁片加密延伸。

## 句法

```
Remove-AzVMDiskEncryptionExtension [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>] [-Force]
 [-NoWait] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 說明
**AzVMDiskEncryptionExtension** Cmdlet 會從虛擬機器中移除磁片加密延伸及相關聯的延伸設定。 如果沒有指定副檔名，這個 Cmdlet 會移除針對執行 Windows 作業系統的虛擬機器或 AzureDiskEncryptionForLinux （針對 Linux 虛擬電腦）的 [預設名稱 AzureDiskEncryption] 的副檔名。 

如果虛擬機器上的加密未事先停用，此 Cmdlet 將會失敗。  若要在虛擬機器上停用加密，請使用 [ [停用-AzVMDiskEncryption](./Disable-AzVMDiskEncryption.md)]。 

## 示例

### 範例1：從虛擬機器移除磁片加密延伸。
```
PS C:\> Remove-AzVMDiskEncryptionExtension -ResourceGroupName "MyResourceGroup" -VMName "MyTestVM"
```

這個命令會針對執行 Windows 作業系統的虛擬機器或以 Linux 為基礎的虛擬機器（名為 MyTestVM）的 AzureDiskEncryptionForLinux，移除具有預設名稱 AzureDiskEncryption 的副檔名。

### 範例2：從虛擬機器移除特定磁片加密延伸。
```
PS C:\> Remove-AzVMDiskEncryptionExtension -ResourceGroupName "MyResourceGroup" -VMName "MyTestVM" -Name "MyDiskEncryptionExtension"
```

這個命令會從名為 MyTestVM 的虛擬機器中移除名為 MyDiskEncryptionExtension 的加密延伸。
