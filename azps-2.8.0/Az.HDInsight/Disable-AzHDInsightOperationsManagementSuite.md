# Disable-AzHDInsightOperationsManagementSuite

## 摘要
在 HDInsight 群集中停用作業管理套件 (OMS) ，而相關的記錄將停止流入在啟用期間指定的 OMS 工作區。

## 句法

```
Disable-AzHDInsightOperationsManagementSuite [-Name] <String> [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 說明
**Disable AzHDInsightOperationsManagementSuite** Cmdlet 會停用 Azure HDInsight 群集中 (OMS) 的 Operations Management Suite。

## 示例

### 範例1
```
PS C:\> Disable-AzHDInsightOMS -Name testcluster

ErrorInfo  :

State      : Succeeded

RequestId  : 1417ad86-d055-48cd-9d68-a5c19a212a3a

StatusCode : OK
```

將會在 HDInsight 群集上停用作業管理套件 (OMS) ，而相關記錄將停止流動至 OMS 工作區。

### 範例2
```
PS C:\> Disable-AzHDInsightOMS -Name testcluster -ResourceGroupName testrg

ErrorInfo  :

State      : Succeeded

RequestId  : 1417ad86-d055-48cd-9d68-a5c19a212a3a

StatusCode : OK
```

將會在 HDInsight 群集上停用作業管理套件 (OMS) ，而相關記錄將停止流動至 OMS 工作區。
