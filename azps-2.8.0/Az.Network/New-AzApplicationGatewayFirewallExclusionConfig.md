# New-AzApplicationGatewayFirewallExclusionConfig

## 摘要
為應用程式閘道 waf 建立新的排除規則清單

## 句法

```
New-AzApplicationGatewayFirewallExclusionConfig -Variable <String> -Operator <String> -Selector <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 說明
**新的-AzApplicationGatewayFirewallExclusionConfig** Cmdlet 為應用程式閘道 waf 新增排除規則清單。

## 示例

### 範例1
```powershell
PS C:\> $exclusion1 = New-AzApplicationGatewayFirewallExclusionConfig -Variable "RequestHeaderNames" -Operator "StartsWith" -Selector "xyz"
```

這個命令會為名為 RequestHeaderNames 的變數及名為 [xyz] 的運算子（名為 StartsWith 及選擇器）建立新的排除規則清單設定。 [排除清單] 設定會儲存在 $exclusion 1。
