# Az FrontDoor 模組
## 說明
本節中的主題檔在 Azure 資源管理器中，將 azure 前門服務的 Azure PowerShell Cmdlet 放在 (ARM) 架構中。 FrontDoor 命名空間中存在 Cmdlet。
