# [Azs] 圖庫. 系統管理模組
## 說明
AzureStack 圖庫系統管理員模組的預覽版本，提供在 marketplace 中管理圖庫專案的功能。

## Azs. 管理員 Cmdlet
### [附加 AzsGalleryItem](Add-AzsGalleryItem.md)
將提供者圖庫專案上傳至儲存空間。

### [AzsGalleryItem](Get-AzsGalleryItem.md)
列出圖庫專案。

### [移除-AzsGalleryItem](Remove-AzsGalleryItem.md)
刪除特定的圖庫專案。
