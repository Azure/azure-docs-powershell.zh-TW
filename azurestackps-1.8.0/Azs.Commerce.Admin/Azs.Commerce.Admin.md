# Azs 系統管理模組
## 說明
AzureStack Commerce 系統管理員模組的預覽版本，提供一種方式來查看整個 AzureStack 戳記的匯總資料使用量。 

## Azs Admin Cmdlet
### [AzsSubscriberUsage](Get-AzsSubscriberUsage.md)
在指定的 timespan 期間取得使用資料。
