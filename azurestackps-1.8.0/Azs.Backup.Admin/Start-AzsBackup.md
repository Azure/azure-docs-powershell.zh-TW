# Start-AzsBackup

## 摘要
備份特定位置。

## 句法

### CreateBackup (預設) 
```
Start-AzsBackup [-ResourceGroupName <String>] [-Location <String>] [-AsJob] [-Force] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

### CreateBackup_FromResourceId
```
Start-AzsBackup -ResourceId <String> [-AsJob] [-Force] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 說明
備份特定位置。

## 示例

### --------------------------範例 1--------------------------
```
Start-AzsBackup
```

啟動 Azure 堆疊備份。
