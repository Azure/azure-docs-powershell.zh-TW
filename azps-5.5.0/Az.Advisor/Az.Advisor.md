# Az. Advisor 模組
## 說明
要與 Azure Advisor 互動的命令。

## Az. Advisor Cmdlet
### [AzAdvisorRecommendation](Get-AzAdvisorRecommendation.md)
取得 Azure Advisor 建議的清單。

### [Disable-AzAdvisorRecommendation](Disable-AzAdvisorRecommendation.md)
停用 Azure Advisor 建議。

### [Enable-AzAdvisorRecommendation](Enable-AzAdvisorRecommendation.md)
啟用 Azure Advisor 建議 (s) 。

### [AzAdvisorConfiguration](Get-AzAdvisorConfiguration.md)
取得指定訂閱或資源群組的 Azure Advisor 設定。

### [Set-AzAdvisorConfiguration](Set-AzAdvisorConfiguration.md)
更新或建立 Azure Advisor 配置。
