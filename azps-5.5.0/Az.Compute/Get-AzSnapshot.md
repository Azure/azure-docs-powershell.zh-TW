# Get-AzSnapshot

## 摘要
取得快照的屬性

## 句法

```
Get-AzSnapshot [[-ResourceGroupName] <String>] [[-SnapshotName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 說明
**AzSnapshot** Cmdlet 會取得快照的屬性。
