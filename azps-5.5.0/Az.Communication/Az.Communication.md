# Az. 通訊模組
## 說明
Microsoft Azure PowerShell：通訊 Cmdlet

## Az. 通訊 Cmdlet
### [AzCommunicationService](Get-AzCommunicationService.md)
取得 CommunicationService 及其屬性。

### [AzCommunicationServiceKey](Get-AzCommunicationServiceKey.md)
取得 CommunicationService 資源的便捷鍵。

### [新-AzCommunicationService](New-AzCommunicationService.md)
建立新的 CommunicationService 或更新現有的 CommunicationService。

### [新-AzCommunicationServiceKey](New-AzCommunicationServiceKey.md)
重新產生 CommunicationService 便捷鍵。
PrimaryKey 與 SecondaryKey 無法在同一時間重新產生。

### [移除-AzCommunicationService](Remove-AzCommunicationService.md)
刪除 CommunicationService 的操作。

### [Set-AzCommunicationServiceNotificationHub](Set-AzCommunicationServiceNotificationHub.md)
將 Azure 通知中樞連結至此通訊服務。

### [更新-AzCommunicationService](Update-AzCommunicationService.md)
更新現有 CommunicationService 的操作。
