# Get-AzPolicyAlias

## 摘要
Get-AzPolicyAlias 檢索及輸出已定義別名並符合指定參數值的 Azure 提供者資源類型。 如果未提供任何參數，則會輸出包含別名的所有提供者資源類型。
-ListAvailable 開關會列出所有相符的資源類型，包括不含別名的那些資源類型來修改此行為。

## 句法

```
Get-AzPolicyAlias [-NamespaceMatch <String>] [-ResourceTypeMatch <String>] [-AliasMatch <String>]
 [-PathMatch <String>] [-ApiVersionMatch <String>] [-LocationMatch <String>] [-ListAvailable]
 [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 說明
**AzPolicyAlias** Cmdlet 會取得原則別名清單。
策略別名是由 Azure Policy 用來參照資源類型屬性。
提供的參數可根據資源類型或其別名的各種屬性來限制清單中的專案。
如果目標字串包含使用不區分大小寫的比較，則指定的 match 值會相符。

## 筆記

* 若要展開別名或任何其他屬性，請將輸出管道 `select -ExpandProperty <property>` 。 例如： `Get-AzPolicyAlias -NamespaceMatch 'Microsoft.Cache' -ApiVersionMatch 'alpha' | select -ExpandProperty Aliases | select -Property Name -ExpandProperty Paths`

* 您可以在輸出中使用其他屬性，並透過將輸出輸送至來顯示 `Format-List` 。 例如： `Get-AzPolicyAlias -NamespaceMatch 'Web' -ResourceTypeMatch site -PathMatch cert | Format-List`
