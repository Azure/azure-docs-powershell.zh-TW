# Get-AzApplicationGatewaySslProfilePolicy

## 摘要
取得應用程式閘道 SSL 設定檔的 SSL 原則。

## 句法

```
Get-AzApplicationGatewaySslProfilePolicy -SslProfile <PSApplicationGatewaySslProfile>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 說明
**AzApplicationGatewaySslProfilePolicy** Cmdlet 會取得應用程式閘道 ssl 設定檔的 SSL 原則。

## 示例

### 範例1
```powershell
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $SslProfile  = Get-AzApplicationGatewaySslProfile -Name "SslProfile01" -ApplicationGateway $AppGw
PS C:\> $sslpolicy = Get-AzApplicationGatewaySslProfilePolicy -SslProfile $SslProfile
```

第一個命令會在名為 ResourceGroup01 的資源群組中，取得名為 ApplicationGateway01 的應用程式閘道，並將它儲存在 $AppGw 變數中。 第二個命令會針對 $AppGw 取得名為 SslProfile01 的 SSL 設定檔，並將它儲存 $SslProfile 變數。 最後一個命令會從 SSL 設定檔中取得 SSL 原則 $SslProfile 並將它儲存在 $sslpolicy 變數中。
