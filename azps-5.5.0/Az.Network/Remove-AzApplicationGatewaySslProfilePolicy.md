# Remove-AzApplicationGatewaySslProfilePolicy

## 簡介
從 Azure 應用程式閘道 SSL 設定檔移除 SSL 策略。

## 語法

```
Remove-AzApplicationGatewaySslProfilePolicy -SslProfile <PSApplicationGatewaySslProfile>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
Cmdlet Remove-AzApplicationGatewaySslProfilePolicy Azure 應用程式閘道 SSL 設定檔移除 SSL 策略。

## 例子

### 範例 1
```powershell
PS C:\>$AppGW = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> $profile  = Get-AzApplicationGatewaySslProfile -Name "Profile01" -ApplicationGateway $AppGw
PS C:\> $profile = Remove-AzApplicationGatewaySslProfilePolicy -SslProfile $profile
```

第一個命令會獲得名為 ResourceGroup01 的資源群組中名為 ApplicationGateway01 的應用程式閘道，並儲存在$AppGw變數。 第二個命令會獲得名稱為 Profile01 的 SSL 設定檔$AppGw並儲存在$profile變數中。 最後一個命令會移除儲存在 $profile 中的 ssl 設定檔$profile。
