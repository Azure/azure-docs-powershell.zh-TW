# Get-AzExpressRoutePortLinkConfig

## 摘要
取得 ExpressRoutePort 連結設定。

## 句法

### ResourceNameParameterSet (預設) 
```
Get-AzExpressRoutePortLinkConfig -ExpressRoutePort <PSExpressRoutePort> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### ResourceIdParameterSet
```
Get-AzExpressRoutePortLinkConfig -ResourceId <String> -ExpressRoutePort <PSExpressRoutePort>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 說明
**AzExpressRoutePortLinkConfig** Cmdlet 會檢索 ExpressRoutePort 連結的設定。

## 示例

### 範例1
```powershell
PS C:\> Get-AzExpressRoutePortLinkConfig -ExpressRoutePort $erport -Name Link1
```

取得 ExpressRoutePort $erport 的 Link1 設定

### 範例2
```powershell
PS C:\> Get-AzExpressRoutePortLinkConfig -ExpressRoutePort $erport -ResourceId $id
```

取得 ExpressRoutePort $erport 中與 ResourceId $id 連結的設定
