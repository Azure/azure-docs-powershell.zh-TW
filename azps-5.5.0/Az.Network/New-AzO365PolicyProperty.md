# New-AzO365PolicyProperty

## 摘要
建立 office 365 流量排列原則物件。

## 句法

```
New-AzO365PolicyProperty [-Allow] [-Optimize] [-Default] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## 說明
建立要與 New-AzVpnSite 與 Update-AzVpnSite Cmdlet 搭配使用的 office 365 專題原則。
## 示例

### 範例1
```powershell
PS C:\> $policy = New-AzO365PolicyProperty -Allow -Optimize
```

建立 office 365 流量排列原則，並允許 [允許] 和 [優化] 類別的流量。
