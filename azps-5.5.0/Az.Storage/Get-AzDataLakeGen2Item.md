# Get-AzDataLakeGen2Item

## 摘要
取得檔案系統中檔案或目錄的詳細資料。

## 句法

```
Get-AzDataLakeGen2Item [-FileSystem] <String> [-Path <String>] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 說明
**AzDataLakeGen2Item** Cmdlet 會取得 Azure 儲存空間帳戶之檔案系統中檔案或目錄的詳細資料。
這個 Cmdlet 只有在已針對儲存空間帳戶啟用階層命名空間時才適用。 此類型的帳戶可使用「-EnableHierarchicalNamespace $true」執行「新-AzStorageAccount」 Cmdlet 來建立。
