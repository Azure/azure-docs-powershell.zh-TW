# Az ADDomainServices 模組
## 說明
Microsoft Azure PowerShell： AdDomainServices Cmdlet

## Az ADDomainServices Cmdlet
### [AzADDomainService](Get-AzADDomainService.md)
[取得網域服務] 作業會檢索網域服務的 json 表示。

### [新-AzADDomainService](New-AzADDomainService.md)
[建立網域服務] 作業會建立具有指定參數的新網域服務。
如果特定服務已存在，則任何 patchable 屬性都會更新，而且任何不可變的屬性都會保持不變。

### [新-AzADDomainServiceForestTrust](New-AzADDomainServiceForestTrust.md)
為 ForestTrust 建立記憶體內建物件

### [新-AzADDomainServiceReplicaSet](New-AzADDomainServiceReplicaSet.md)
為 ReplicaSet 建立記憶體內建物件

### [移除-AzADDomainService](Remove-AzADDomainService.md)
[刪除網域服務] 作業會刪除現有的網域服務。

### [更新-AzADDomainService](Update-AzADDomainService.md)
更新網域服務操作可用來更新現有的部署。
更新呼叫只支援修補程式本文中所列的屬性。
