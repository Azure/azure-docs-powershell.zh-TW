# Az KubernetesConfiguration 模組
## 說明
Microsoft Azure PowerShell： KubernetesConfiguration Cmdlet

## Az KubernetesConfiguration Cmdlet
### [AzKubernetesConfiguration](Get-AzKubernetesConfiguration.md)
取得原始程式碼管理設定的詳細資料。

### [新-AzKubernetesConfiguration](New-AzKubernetesConfiguration.md)
建立新的 Kubernetes 來源控制項配置。

### [移除-AzKubernetesConfiguration](Remove-AzKubernetesConfiguration.md)
這將會刪除用來設定來源控制設定的 YAML 檔案，從而停止未來與來源存放庫的同步處理。
