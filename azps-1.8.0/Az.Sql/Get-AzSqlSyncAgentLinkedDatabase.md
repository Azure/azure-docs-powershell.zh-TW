# Get-AzSqlSyncAgentLinkedDatabase

## 摘要
傳回同步處理代理程式所連結之 SQL Server 資料庫的相關資訊。

## 句法

```
Get-AzSqlSyncAgentLinkedDatabase [-ServerName] <String> [-SyncAgentName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 說明
**AzSqlSyncAgentLinkedDatabases** Cmdlet 會傳回同步處理代理程式所連結之 SQL Server 資料庫的相關資訊。

## 示例

### 範例1：取得 Azure SQL 同步處理代理程式的連結 SQL Server 資料庫。
```
PS C:\> Get-AzSqlSyncAgentLinkedDatabases -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -SyncAgentName "SyncAgent01" | Format-List
SeverName                 : sever01
DatabaseId                : databaseId
DatabaseName              : database01
DatabaseType              : SQLServerDatabase
Description               : 
UserName                  : myAccount
```

這個命令會傳回由 Azure SQL 同步處理代理程式連結的連結 SQL Server 資料庫。
