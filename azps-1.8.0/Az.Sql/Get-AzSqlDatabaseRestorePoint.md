# Get-AzSqlDatabaseRestorePoint

## 摘要
檢索可還原 SQL 資料倉儲的不同還原點。

## 句法

```
Get-AzSqlDatabaseRestorePoint [-ServerName] <String> [-DatabaseName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 說明
**AzSqlDatabaseRestorePoint** Cmdlet 會檢索 Azure SQL 資料倉儲可以還原的不同還原點。
針對 Azure SQL 資料庫，[還原] 視窗是連續的。
這表示資料庫備份保留期間中的任何時間點都可以用來做為還原點。
Azure 上的 SQL Server Stretch Database 服務也支援此 Cmdlet。

## 示例

### 範例1：取得所有還原點
```
PS C:\>Get-AzSqlDatabaseRestorePoint -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
ResourceGroupName        : resourcegroup01
ServerName               : server01
DatabaseName             : database01
Location                 : Central US
RestorePointType         : CONTINUOUS
RestorePointCreationDate : 
EarliestRestoreDate      : 8/12/2015 12:00:00 AM
RestorePointLabel        : RestorePoint01
```

這個命令會傳回 Azure SQL Database （名為 Database01）所有可用的還原點。
