# Remove-AzSqlServerThreatDetectionPolicy

## 摘要
從伺服器移除威脅偵測原則。

## 句法

```
Remove-AzSqlServerThreatDetectionPolicy [-PassThru] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 說明
Remove-AzSqlServerThreatDetectionPolicy Cmdlet 會從 Azure SQL server 中移除威脅偵測原則。
若要使用這個 Cmdlet，請指定 ResourceGroupName 和 ServerName 參數來識別這個 Cmdlet 移除原則的伺服器。

## 示例

### 範例1：移除資料庫的威脅偵測原則
```
PS C:\> Remove-AzSqlServerThreatDetectionPolicy -ResourceGroupName "ResourceGroup11" -ServerName "Server01"
```

這個命令會從名為 Server01 的伺服器中移除威脅偵測原則。
