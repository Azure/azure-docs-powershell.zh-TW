# Register-AzModule

## 摘要
僅供內部使用-提供 AutoRest 產生的 Cmdlet 的執行時間支援

## 句法

```
Register-AzModule [<CommonParameters>]
```

## 說明
僅供內部使用-提供 AutoRest 產生的 Cmdlet 的執行時間支援

## 示例

### 範例1
```powershell
PS C:\> Register-AzModule
```

由 AutoRest 產生的 Cmdlet 在內部使用
