# New-AzRedisCacheScheduleEntry

## 摘要
建立排程專案。

## 句法

```
New-AzRedisCacheScheduleEntry -DayOfWeek <String> -StartHourUtc <Int32> [-MaintenanceWindow <TimeSpan>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 說明
**新的-AzRedisCacheScheduleEntry** Cmdlet 會建立 **PSScheduleEntry** 物件。
Azure Redis Cache 修補程式排程 Cmdlet （例如 New-AzRedisCachePatchSchedule Cmdlet）需要排程專案物件。

## 示例

### 範例1：建立週末的排程專案
```
PS C:\>New-AzRedisCacheScheduleEntry -DayOfWeek "Weekend" -StartHourUtc 2 -MaintenanceWindow "06:00:00"
```

這個命令會建立一個代表有指定開始時間和視窗之週末排程的 **PSScheduleEntry** 物件。
