# Remove-AzBatchJobSchedule

## 摘要
移除批次作業排程。

## 句法

```
Remove-AzBatchJobSchedule [-Id] <String> [-Force] -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 說明
**AzBatchJobSchedule** Cmdlet 會移除 Azure 批次作業排程。

## 示例

### 範例1：刪除批次作業排程
```
PS C:\>Remove-AzBatchJobSchedule -Id "MyJobSchedule" -BatchContext $Context
```

這個命令會刪除具有識別碼 MyJobSchedule 的作業排程。
命令會在刪除工作之前提示您進行確認。
使用 Get-AzBatchAccountKeys Cmdlet 將內容指派給 $CoNtext 變數。

### 範例2：不需使用管線確認即可刪除批次作業
```
PS C:\>Get-AzBatchJobSchedule -Id "MyJobSchedule" -BatchContext $Context | Remove-AzBatchJobSchedule -Force -BatchContext $Context
```

這個命令會透過使用 Get-AzBatchJobSchedule Cmdlet 來取得 ID 為 MyJobSchedule 的作業排程。
透過使用管線運算子，命令會將作業排程傳送至目前的 Cmdlet。
該命令會刪除該作業排程。
因為命令包含 *Force* 參數，所以它不會提示您進行確認。
