# Get-AzApplicationSecurityGroup

## 摘要
取得應用程式安全性群組。

## 句法

```
Get-AzApplicationSecurityGroup [-ResourceGroupName <String>] [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 說明
**AzApplicationSecurityGroup** Cmdlet 會取得應用程式安全性群組。
