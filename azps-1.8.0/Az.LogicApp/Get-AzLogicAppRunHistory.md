# Get-AzLogicAppRunHistory

## 摘要
取得邏輯 app 的執行歷程記錄。

## 句法

```
Get-AzLogicAppRunHistory -ResourceGroupName <String> -Name <String> [-RunName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 說明
**AzLogicAppRunHistory** Cmdlet 會取得邏輯 app 的執行歷程記錄。
這個 Cmdlet 會傳回 **WorkflowRun** 物件的集合。
指定邏輯 app 與資源群組。
此模組支援動態參數。
若要使用動態參數，請在命令中輸入。
若要探索動態參數的名稱，請在 Cmdlet 名稱後輸入連字號 ( ) ，然後重複按 Tab 鍵以迴圈顯示可用的參數。
如果您省略必要的範本參數，則 Cmdlet 會提示您輸入值。
