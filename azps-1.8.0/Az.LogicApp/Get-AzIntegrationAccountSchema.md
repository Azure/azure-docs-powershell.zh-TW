# Get-AzIntegrationAccountSchema

## 摘要
取得整合帳戶架構。

## 句法

```
Get-AzIntegrationAccountSchema [-ResourceGroupName <String>] [-Name <String>] [-SchemaName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 說明
**AzIntegrationAccountSchema** Cmdlet 會取得整合帳戶架構。
指定整合帳戶名稱、資源群組名稱和架構名稱。
此模組支援動態參數。
若要使用動態參數，請在命令中輸入。
若要探索動態參數的名稱，請在 Cmdlet 名稱後輸入連字號 ( ) ，然後重複按 Tab 鍵以迴圈顯示可用的參數。
如果您省略必要的範本參數，則 Cmdlet 會提示您輸入值。
