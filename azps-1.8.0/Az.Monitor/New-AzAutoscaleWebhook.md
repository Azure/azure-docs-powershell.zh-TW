# New-AzAutoscaleWebhook

## 摘要
建立自動縮放 webhook。

## 句法

```
New-AzAutoscaleWebhook [-ServiceUri] <String> [[-Property] <Hashtable>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 說明
**新的-AzAutoscaleWebhook** Cmdlet 會建立自動縮放 webhook。
