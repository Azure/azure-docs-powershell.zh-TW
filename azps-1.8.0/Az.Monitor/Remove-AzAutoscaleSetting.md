# Remove-AzAutoscaleSetting

## 摘要
移除自動縮放設定。

## 句法

```
Remove-AzAutoscaleSetting -ResourceGroupName <String> -Name <String> [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 說明
**AzAutoscaleSetting** Cmdlet 會移除自動縮放設定。
您必須指定設定的名稱，以及其所指派之資源群組的名稱。
這個 Cmdlet 會實現 ShouldProcess 模式，亦即，在實際建立、修改或移除資源之前，它可能會要求使用者進行確認。
