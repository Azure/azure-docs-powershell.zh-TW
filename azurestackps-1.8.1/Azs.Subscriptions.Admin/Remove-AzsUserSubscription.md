# Remove-AzsUserSubscription

## 摘要
移除指定的租使用者訂閱。

## 句法

```
Remove-AzsUserSubscription [-SubscriptionId] <Guid> [-Force] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 說明
移除指定的租使用者訂閱。

## 示例

### --------------------------範例 1--------------------------
```
Remove-AzsUserSubscription -SubscriptionId "c90173b1-de7a-4b1d-8600-b832b0e65946"
```

移除指定的租使用者訂閱。
