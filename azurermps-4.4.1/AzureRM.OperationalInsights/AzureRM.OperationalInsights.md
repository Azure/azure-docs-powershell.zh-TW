# AzureRM OperationalInsights Module
## 說明
本主題顯示適用于 Azure Operational Insights Cmdlet 的說明主題。

## AzureRM OperationalInsights Cmdlet
### [Disable-AzureRmOperationalInsightsIISLogCollection](Disable-AzureRmOperationalInsightsIISLogCollection.md)
停止從電腦收集 IIS 記錄。

### [Disable-AzureRmOperationalInsightsLinuxCustomLogCollection](Disable-AzureRmOperationalInsightsLinuxCustomLogCollection.md)
停止從 Linux 電腦收集自訂記錄。

### [Disable-AzureRmOperationalInsightsLinuxPerformanceCollection](Disable-AzureRmOperationalInsightsLinuxPerformanceCollection.md)
停止從 Linux 電腦收集效能計數器。

### [Disable-AzureRmOperationalInsightsLinuxSyslogCollection](Disable-AzureRmOperationalInsightsLinuxSyslogCollection.md)
停止從 Linux 電腦收集 syslog 資料。

### [Enable-AzureRmOperationalInsightsIISLogCollection](Enable-AzureRmOperationalInsightsIISLogCollection.md)
從工作區中的電腦開始收集 IIS 記錄。

### [Enable-AzureRmOperationalInsightsLinuxCustomLogCollection](Enable-AzureRmOperationalInsightsLinuxCustomLogCollection.md)
從 Linux 電腦開始收集自訂記錄。

### [Enable-AzureRmOperationalInsightsLinuxPerformanceCollection](Enable-AzureRmOperationalInsightsLinuxPerformanceCollection.md)
從 Linux 電腦開始收集效能計數器。

### [Enable-AzureRmOperationalInsightsLinuxSyslogCollection](Enable-AzureRmOperationalInsightsLinuxSyslogCollection.md)
從 Linux 電腦開始收集 syslog 資料。

### [AzureRmOperationalInsightsDataSource](Get-AzureRmOperationalInsightsDataSource.md)
在 Azure Log Analytics 工作區中取得資料來源。

### [AzureRmOperationalInsightsIntelligencePacks](Get-AzureRmOperationalInsightsIntelligencePacks.md)
取得可用的智慧套件。

### [AzureRmOperationalInsightsLinkTargets](Get-AzureRmOperationalInsightsLinkTargets.md)
取得不與訂閱相關聯的帳戶。

### [AzureRmOperationalInsightsSavedSearch](Get-AzureRmOperationalInsightsSavedSearch.md)
傳回指定工作區的所有已儲存的搜尋。

### [AzureRmOperationalInsightsSavedSearchResults](Get-AzureRmOperationalInsightsSavedSearchResults.md)
傳回查詢的結果。

### [AzureRmOperationalInsightsSchema](Get-AzureRmOperationalInsightsSchema.md)
傳回與工作區相關聯的架構。

### [AzureRmOperationalInsightsSearchResults](Get-AzureRmOperationalInsightsSearchResults.md)
根據指定的參數傳回搜尋結果。

### [AzureRmOperationalInsightsStorageInsight](Get-AzureRmOperationalInsightsStorageInsight.md)
取得儲存空間洞察力的相關資訊。

### [AzureRmOperationalInsightsWorkspace](Get-AzureRmOperationalInsightsWorkspace.md)
取得工作區的相關資訊。

### [AzureRmOperationalInsightsWorkspaceManagementGroups](Get-AzureRmOperationalInsightsWorkspaceManagementGroups.md)
取得連接至工作區之管理群組的詳細資料。

### [AzureRmOperationalInsightsWorkspaceSharedKeys](Get-AzureRmOperationalInsightsWorkspaceSharedKeys.md)
取得工作區的共用金鑰。

### [AzureRmOperationalInsightsWorkspaceUsage](Get-AzureRmOperationalInsightsWorkspaceUsage.md)
取得工作區的使用資料。

### [新-AzureRmOperationalInsightsAzureActivityLogDataSource](New-AzureRmOperationalInsightsAzureActivityLogDataSource.md)
從指定的訂閱收集 Azure 活動記錄。

### [新-AzureRmOperationalInsightsComputerGroup](New-AzureRmOperationalInsightsComputerGroup.md)
建立電腦群組。

### [新-AzureRmOperationalInsightsCustomLogDataSource](New-AzureRmOperationalInsightsCustomLogDataSource.md)
定義自訂的日誌收集原則。

### [新-AzureRmOperationalInsightsLinuxPerformanceObjectDataSource](New-AzureRmOperationalInsightsLinuxPerformanceObjectDataSource.md)
新增效能計數器至工作區中的所有 Linux 電腦。

### [新-AzureRmOperationalInsightsLinuxSyslogDataSource](New-AzureRmOperationalInsightsLinuxSyslogDataSource.md)
新增資料來源至 Linux 電腦。

### [新-AzureRmOperationalInsightsSavedSearch](New-AzureRmOperationalInsightsSavedSearch.md)
使用指定的參數建立新的已儲存搜尋。

### [新-AzureRmOperationalInsightsStorageInsight](New-AzureRmOperationalInsightsStorageInsight.md)
在工作區中建立儲存洞察力。

### [新-AzureRmOperationalInsightsWindowsEventDataSource](New-AzureRmOperationalInsightsWindowsEventDataSource.md)
從執行 Windows 作業系統的電腦收集事件記錄。

### [新-AzureRmOperationalInsightsWindowsPerformanceCounterDataSource](New-AzureRmOperationalInsightsWindowsPerformanceCounterDataSource.md)
新增執行 Windows 作業系統之已連接的電腦的 Windows 效能計數器資料來源。

### [新-AzureRmOperationalInsightsWorkspace](New-AzureRmOperationalInsightsWorkspace.md)
建立工作區。

### [移除-AzureRmOperationalInsightsDataSource](Remove-AzureRmOperationalInsightsDataSource.md)
刪除資料來源。

### [移除-AzureRmOperationalInsightsSavedSearch](Remove-AzureRmOperationalInsightsSavedSearch.md)
從工作區移除已儲存的搜尋。

### [移除-AzureRmOperationalInsightsStorageInsight](Remove-AzureRmOperationalInsightsStorageInsight.md)
移除儲存空間洞察力。

### [移除-AzureRmOperationalInsightsWorkspace](Remove-AzureRmOperationalInsightsWorkspace.md)
移除工作區。

### [Set-AzureRmOperationalInsightsDataSource](Set-AzureRmOperationalInsightsDataSource.md)
更新資料來源。

### [Set-AzureRmOperationalInsightsIntelligencePack](Set-AzureRmOperationalInsightsIntelligencePack.md)
啟用或停用指定的智慧包。

### [Set-AzureRmOperationalInsightsSavedSearch](Set-AzureRmOperationalInsightsSavedSearch.md)
更新已存在的已儲存搜尋。

### [Set-AzureRmOperationalInsightsStorageInsight](Set-AzureRmOperationalInsightsStorageInsight.md)
更新儲存空間洞察力。

### [Set-AzureRmOperationalInsightsWorkspace](Set-AzureRmOperationalInsightsWorkspace.md)
更新工作區。
