# Stop-AzureBatchJob

## 摘要
停止批次作業。

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 句法

```
Stop-AzureBatchJob [-Id] <String> [[-TerminateReason] <String>] -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 說明
**AzureBatchJob** Cmdlet 會停止 Azure 批次作業。
這個命令會將工作標示為已完成。

## 示例

### 範例1：停止批次作業
```
PS C:\>Stop-AzureBatchJob -Id "Job-000001" -TerminateReason "No more tasks to run" -BatchContext $Context
```

這個命令會停止具有識別碼作業的作業-000001。
該命令會指定您選擇停止作業的原因。
使用 Get-AzureRmBatchAccountKeys Cmdlet 將內容指派給 $CoNtext 變數。
