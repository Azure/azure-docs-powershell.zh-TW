# AzureRM 帳單模組
## 說明
本節中的主題會在 Azure 資源管理器中，將 azure 帳單的 Azure PowerShell Cmdlet 記錄在 (ARM) 架構中。 Cmdlet 會存在於 Microsoft Azure. 帳單命名空間中。

## AzureRM 帳單 Cmdlet
### [AzureRmBillingInvoice](Get-AzureRmBillingInvoice.md)
取得訂閱的帳單發票。

### [AzureRmBillingPeriod](Get-AzureRmBillingPeriod.md)
取得訂閱的計費期間。
