# Remove-AzureRmSqlServerKeyVaultKey

## 摘要
從 SQL server 移除金鑰保存庫金鑰。

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 句法

```
Remove-AzureRmSqlServerKeyVaultKey [-KeyId] <String> [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 說明
Remove-AzureRmSqlServerKeyVaultKey Cmdlet 會從指定的 SQL server 中移除金鑰保存庫金鑰。

請注意，不會變更 SQL server 對金鑰電子倉庫的許可權。
若要變更許可權，請使用 [設定-AzureRmKeyVaultAccessPolicy]。

請注意，這個 Cmdlet 不會變更金鑰保存庫。
若要從 [主要電子倉庫] 移除金鑰，請使用 [移除-AzureKeyVaultKey]。

## 示例

### --------------------------範例1：移除金鑰保存庫金鑰--------------------------
```
PS C:\> Remove-AzureRmSqlServerKeyVaultKey -KeyId 'https://contoso.vault.azure.net/keys/contosokey/01234567890123456789012345678901' -ServerName 'ContosoServer' -ResourceGroupName 'ContosoResourceGroup'
```

這個命令會從指定的伺服器移除 Id 為 '」的主要保存庫金鑰 https://contoso.vault.azure.net/keys/contosokey/01234567890123456789012345678901 。
