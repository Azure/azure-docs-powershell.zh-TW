# Set-AzAdvisorConfiguration

## 簡介
更新或建立 Azure 建議程式組配置。

## 語法

### InputObjectRgExcludeParameterSet (預設) 
```
Set-AzAdvisorConfiguration [-Exclude] [[-ResourceGroupName] <String>]
 [[-InputObject] <PsAzureAdvisorConfigurationData>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### InputObjectLowObcludeParameterSet 
```
Set-AzAdvisorConfiguration [-Exclude] [-LowCpuThreshold] <Int32>
 [[-InputObject] <PsAzureAdvisorConfigurationData>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## 描述
用來更新 Azure Advisor 的組配置。 有兩種類型的組組：訂閱層級組和 ResourceGroup 層級組組。 

訂閱層級組式：此類型的訂閱只能有一個組式。 您可以使用此 Cmdlet 更新 LowCmThreshold 和 Exclude 屬性。
ResourceGroup 層級組配置：每個 ResourceGroup 只能有一個組組。 只能使用此 Cmdlet 更新 Exclude 屬性。
