# Get-AzAdvisorConfiguration

## 簡介
取得給定訂閱或資源群組的 Azure 建議程式組。

## 語法

```
Get-AzAdvisorConfiguration [-ResourceGroupName <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## 描述
與訂閱相關聯的組配置有兩種類型：

訂閱層級組式：訂閱只能有一種這種類型的組式。 Low 是否擁有和排除是此類型組式的唯一屬性。

ResourceGroup 層級組配置：訂閱中每個 ResourceGroup 只能有一個組組。 排除是此類型組式的唯一屬性。
