# Remove-AzKeyVaultManagedHsm

## 簡介
刪除受管理的 HSM。

## 語法

### RemoveManaged4mByName (預設) 
```
Remove-AzKeyVaultManagedHsm [-Name] <String> [[-ResourceGroupName] <String>] [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveManaged1mByInputObject
```
Remove-AzKeyVaultManagedHsm [-InputObject] <PSManagedHsm> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### RemoveManagedManagedManmByResourceId
```
Remove-AzKeyVaultManagedHsm [-ResourceId] <String> [-Force] [-AsJob] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
**Remove-AzKeyVaultManagedPvm** Cmdlet 會刪除指定的 Managed HSM。
它也會刪除該實例中包含的所有金鑰。
請注意，雖然指定資源群組是這個 Cmdlet 的選擇性專案，但為了提升績效，您應該這麼做。

## 例子

### 範例 1：移除受管理的 HSM
```powershell
PS C:\> Remove-AzKeyVaultManagedHsm -HsmName 'myhsm' -Force

True
```

此命令會從您目前的訂閱移除名為 mym 的受管理 hsm。

### 範例 2：從指定的資源群組移除受管理的 hsm
```powershell
PS C:\> Remove-AzKeyVaultManagedHsm -HsmName 'myhsm' -ResourceGroupName "myrg1" -PassThru

True
```

此命令會從名為 myrg1 的資源群組移除名為 mym 的受管理 hsm。
如果您未指定資源組名，Cmdlet 會搜尋您目前訂閱中要刪除的受管理 HSM。
