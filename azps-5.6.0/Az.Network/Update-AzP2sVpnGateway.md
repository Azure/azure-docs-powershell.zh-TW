# Update-AzP2sVpnGateway

## 簡介
在 VirtualHub 下更新現有的 P2S VpnGateway，以指向網站連接。

## 描述
**Update-AzP2s VpnGateway** Cmdlet 可讓您使用新的 VpnClientAddressPool 或新的 VpnServerConfiguration 或 VpnGatewayScaleUnit 變更，在 VirtualHub 下更新現有的 P2S VpnGateway。
