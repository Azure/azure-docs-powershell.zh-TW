# Remove-AzDdosProtectionPlan

## 簡介
移除 DDoS 保護計劃。

## 語法

```
Remove-AzDdosProtectionPlan -ResourceGroupName <String> -Name <String> [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
Cmdlet Remove-AzDdosProtectionPlan移除 DDoS 保護計劃。

## 例子

### 範例 1：移除空白的 DDoS 保護計劃
```
D:\> Remove-AzDdosProtectionPlan -ResourceGroupName ResourceGroupName -Name DdosProtectionPlan
```

在此例中，我們會移除指定的 DDoS 保護方案。

### 範例 2：移除與虛擬網路相關聯的 DDoS 保護計劃
```
D:\> $vnet = Get-AzVirtualNetwork -Name VnetName -ResourceGroupName ResourceGroupName
D:\> $vnet.DdosProtectionPlan = $null
D:\> $vnet.EnableDdosProtection = $false
D:\> $vnet | Set-AzVirtualNetwork
```

如果 DDoS 保護方案與虛擬網路相關聯，就無法刪除。 因此，第一個步驟是取消兩個物件的關聯。 在這裡，我們會取得與計畫關聯的最新版本的虛擬網路，並且將 **屬性 DdosProtectionPlan** 設定為空白值，而標有 **EnableDdosProtection** (此標號若沒有計劃) 就無法成立。
接著，我們將本地變數放入 **Set-AzVirtualNetwork** 中，以持續新的狀態。 此時，計畫不再與虛擬網路關聯。
如果這是最後一個與計畫相關聯的計畫，可以使用 Remove-AzDdosProtectionPlan 命令移除 DDoS 保護計劃。
