# Get-AzVirtualNetworkGatewayAdvertisedRoute

## 簡介
列出 Azure 虛擬網路閘道所宣告的路由

## 語法

```
Get-AzVirtualNetworkGatewayAdvertisedRoute -VirtualNetworkGatewayName <String> -ResourceGroupName <String>
 -Peer <String> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 描述
如果具有 BGP 對等的 IP，請列舉由指定的 Azure 虛擬網路閘道向該對等電腦宣告的路由。 

## 例子

### 範例 1
```
PS C:\> Get-AzVirtualNetworkGatewayAdvertisedRoute -VirtualNetworkGatewayName gatewayName -ResourceGroupName resourceGroupName -Peer 10.0.0.254
```

針對在資源群組 resourceGroupName 中名為 gatewayName 的 Azure 閘道，會以 IP 10.0.0.254，從 BGP 對等程式中，取回要宣告至 BGP 對等的路由清單。

### 範例 2
```
PS C:\> $bgpPeerStatus = Get-AzVirtualNetworkGatewayBGPPeerStatus -VirtualNetworkGatewayName gatewayName -ResourceGroupName resourceGroupName
PS C:\> Get-AzVirtualNetworkGatewayAdvertisedRoute -VirtualNetworkGatewayName gatewayName -ResourceGroupName resourceGroupName -Peer $bgpPeerStatus[0].Neighbor
```

針對資源群組 resourceGroupName 中名為 gatewayName 的 Azure 閘道，會從閘道的 BGP 對等程式清單中，將宣告至第一個 BGP 對等的路由進行取用。
