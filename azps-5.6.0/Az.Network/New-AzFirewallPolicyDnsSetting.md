# New-AzFirewallPolicyDnsSetting

## 簡介
為 Azure 防火牆政策建立新的 DNS 設定

## 語法

```
New-AzFirewallPolicyDnsSetting [-EnableProxy] [-Server <String[]>] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
**New-AzFirewallPolicyDnsSetting** Cmdlet 會為 Azure 防火牆策略建立 DNS 設定物件

## 例子

### 1. 建立空白政策
```powershell
PS C:\> New-AzFirewallPolicyDnsSetting -EnableProxy
```

此範例會建立設定啟用 dns Proxy 的 dns 設定物件。

### 2. 使用 ThreatIntel 模式建立空白策略
```powershell
PS C:\> $dnsServers = @("10.10.10.1", "20.20.20.2")
PS C:\> New-AzFirewallPolicyDnsSetting -EnableProxy -Server $dnsServers
```

此範例會建立設定啟用 dns Proxy 和設定自訂 dns 伺服器的 dns 設定物件。
