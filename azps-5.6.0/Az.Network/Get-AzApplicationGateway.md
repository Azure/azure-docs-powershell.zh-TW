# Get-AzApplicationGateway

## 簡介
獲得應用程式閘道。

## 語法

```
Get-AzApplicationGateway [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 描述
**Get-AzApplicationGateway** Cmdlet 會取得應用程式閘道。
