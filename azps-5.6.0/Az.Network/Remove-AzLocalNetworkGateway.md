# Remove-AzLocalNetworkGateway

## 簡介
刪除本地網路閘道

## 語法

```
Remove-AzLocalNetworkGateway -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
Local Network Gateway 是代表您的 VPN 裝置內部部署的物件。
**Remove-AzLocalNetworkGateway** Cmdlet 會根據名稱和資源組名，刪除代表您 on-prem 閘道的物件。

## 例子

### 範例 1：刪除本地網路閘道
```powershell
Remove-AzLocalNetworkGateway -Name myLocalGW -ResourceGroupName myRG
```

刪除資源群組 "myLocalGW" 中名稱為 "myLocalGW" 的 Local Network Gateway 物件注意：您必須先刪除使用 **Remove-AzVirtualNetworkGatewayConnection** Cmdlet 到 Local Network 閘道的所有連結。
