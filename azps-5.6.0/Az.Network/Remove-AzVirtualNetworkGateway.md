# Remove-AzVirtualNetworkGateway

## 簡介
刪除虛擬網路閘道

## 語法

```
Remove-AzVirtualNetworkGateway -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
虛擬網路閘道是代表 Azure 中閘道的物件。
**Get-AzVirtualNetworkGateway** Cmdlet 會根據名稱和資源組名，在 Azure 中會返回閘道的物件。

## 例子

### 範例 1：刪除虛擬網路閘道
```powershell
Remove-AzVirtualNetworkGateway -Name myGateway -ResourceGroupName myRG
```

刪除資源群組「myRG」內名稱為「myGateway」的虛擬網路閘道物件：您必須先刪除使用 **Remove-AzVirtualNetworkGatewayConnection** Cmdlet 之虛擬網路閘道的所有連結。
