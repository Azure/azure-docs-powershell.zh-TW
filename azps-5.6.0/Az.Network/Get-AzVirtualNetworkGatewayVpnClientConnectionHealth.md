# Get-AzVirtualNetworkGatewayVpnClientConnectionHealth

## 簡介 
取得每個 vpn 用戶端連接的 Azure 虛擬網路閘道 vpn 用戶端連接健康狀態清單

## 語法

```
Get-AzVirtualNetworkGatewayVpnClientConnectionHealth -ResourceName <String> -ResourceGroupName <String> -InputObject <PSVirtualNetworkGateway> -ResourceId <ResourceId>
 [-AsJob] [<CommonParameters>]
```

## 描述
列出所有已連接的 vpn 用戶端連接健康狀態。 這包括：vpnConnectionId vpnConnectionDuration vpnConnectionTime publicIpAddress privateIpAddress vpnUserName maxBandwidth 出口PacketsTransferred出口BytesTransferred ingressPacketsTransferred ingressBytesTransferred maxPacketsPerRation
