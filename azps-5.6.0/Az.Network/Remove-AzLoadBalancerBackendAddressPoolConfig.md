# Remove-AzLoadBalancerBackendAddressPoolConfig

## 簡介
從負載平衡器移除後端位址資料庫組。

## 語法

```
Remove-AzLoadBalancerBackendAddressPoolConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
**Remove-AzLoadBalancerBackendAddressPool Cmdlet** 會從負載平衡器移除後端位址集區。

## 例子

### 範例 1：從負載平衡器移除後端位址資料庫組
```
PS C:\>Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup" | Remove-AzLoadBalancerBackendAddressPoolConfig -Name "BackendAddressPool02" | Set-AzLoadBalancer
```

此命令會取得名為 MyLoadBalancer 的負載平衡器，並傳遞至 **Remove-AzLoadBalancerBackendAddressPoolConfig，** 這會從 MyLoadBalancer 移除後端AddressPool02 組式。
最後，Set-AzLoadBalancer Cmdlet 會更新 MyLoadBalancer。
請注意，後端位址資料庫組必須先存在，才能將其刪除。
