# Get-AzLoadBalancerOutboundRuleConfig

## 簡介
取得負載平衡器中的外發規則組式。

## 語法

```
Get-AzLoadBalancerOutboundRuleConfig -LoadBalancer <PSLoadBalancer> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 描述
**Get-AzLoadBalancerOutboundRuleConfig** Cmdlet 會取得輸出規則組式，或負載平衡器中的輸出規則組式清單。

## 例子

### 範例 1：在負載平衡器中取得外發規則組式
```powershell
PS C:\>$slb = Get-AzLoadBalancer -ResourceGroupName "MyResourceGroup" -Name "MyLoadBalancer"
PS C:\>Get-AzLoadBalancerOutboundRuleConfig -LoadBalancer $slb -Name "MyRule"
```

第一個命令會取得名為 MyLoadBalancer 的負載平衡器，然後將它儲存在變數 $slb。
第二個命令會取得名為 MyRule 的外發規則組式，該規則組與負載平衡器相關聯。
