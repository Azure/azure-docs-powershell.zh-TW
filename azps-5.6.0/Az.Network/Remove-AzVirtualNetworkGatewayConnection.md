# Remove-AzVirtualNetworkGatewayConnection

## 簡介
刪除虛擬網路閘道連接

## 語法

```
Remove-AzVirtualNetworkGatewayConnection -Name <String> -ResourceGroupName <String> [-Force] [-PassThru]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
虛擬網路閘道連接是代表 IPsec 服務物件 (網站對網站或 Vnet-to-Vnet) 連接至 Azure 中的虛擬網路閘道。
**Remove-AzVirtualNetworkGatewayConnection** Cmdlet 會根據您的名稱和資源組名刪除您的連線物件。

## 例子

### 範例 1：刪除虛擬網路閘道連接
```powershell
Remove-AzVirtualNetworkGatewayConnection -Name myTunnel -ResourceGroupName myRG
```

刪除資源群組 "myRG" 中名稱為 "myTun圳" 的虛擬網路閘道連線物件
