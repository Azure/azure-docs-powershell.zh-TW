# New-AzApplicationGatewayConnectionDraining

## 簡介
為後端 HTTP 設定建立新的連接耗用設定。

## 語法

```
New-AzApplicationGatewayConnectionDraining -Enabled <Boolean> -DrainTimeoutInSec <Int32>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 描述
**New-AzApplicationGatewayConnectionDraining** Cmdlet 為後端 HTTP 設定建立新的連接消耗設定。

## 例子

### 範例 1
```
PS C:\> $connectionDraining = New-AzApplicationGatewayConnectionDraining -Enabled $True -DrainTimeoutInSec 42
```

此命令會建立新的連接耗用設定，將 Enabled 設定為 True 且 DrainTimeoutInSec 設定為 42 秒，並儲存于 $connectionDraining。
