# New-AzPublicIpTag

## 簡介
建立 IP 標記。

## 語法

```
New-AzPublicIpTag -IpTagType <String> -Tag <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## 描述
**New-AzPublicIpTag** Cmdlet 會建立 IP 標記。

## 例子

### 範例 1：建立新 IP 標記
```powershell
$ipTag = New-AzPublicIpTag -IpTagType $ipTagType -Tag $tag
```

此命令會使用標記類型建立新 IP 標記，例如"FirstPartyUsage"，以及類似 "/Sql" 的標記。 這會在 publicIpAddress 建立中使用這些特定標記來配置。
