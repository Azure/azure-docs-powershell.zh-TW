# Get-AzExpressRouteCircuit

## 簡介
從 Azure 獲得 Azure ExpressRoute 回路。

## 語法

```
Get-AzExpressRouteCircuit [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 描述
**Get-AzExpressRouteCircuit** Cmdlet 可用來從訂閱中取回 ExpressRoute 回路物件。 所返回的回路物件可以用來做為在 ExpressRoute 回路上運作之其他 Cmdlet 的輸入。
