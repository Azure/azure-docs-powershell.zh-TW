# Set-AzMigrateServerReplication

## 簡介
更新複製伺服器的目標屬性。

## 語法

### ByIDVCbt (預設) 
```
Set-AzMigrateServerReplication -TargetObjectID <String> [-NicToUpdate <IVMwareCbtNicInput[]>]
 [-SubscriptionId <String>] [-TargetAvailabilitySet <String>] [-TargetAvailabilityZone <String>]
 [-TargetBootDiagnosticsStorageAccount <String>] [-TargetNetworkId <String>] [-TargetResourceGroupID <String>]
 [-TargetVMName <String>] [-TargetVMSize <String>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```

### ByInputObjectVCbt
```
Set-AzMigrateServerReplication -InputObject <IMigrationItem> [-NicToUpdate <IVMwareCbtNicInput[]>]
 [-SubscriptionId <String>] [-TargetAvailabilitySet <String>] [-TargetAvailabilityZone <String>]
 [-TargetBootDiagnosticsStorageAccount <String>] [-TargetNetworkId <String>] [-TargetResourceGroupID <String>]
 [-TargetVMName <String>] [-TargetVMSize <String>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## 描述
Cmdlet Set-AzMigrateServerReplication更新複製伺服器的目標屬性。

## 筆記

別名

複雜的參數屬性

若要建立下列描述的參數，請建構包含適當屬性的雜湊表。 有關雜湊表的資訊，請執行Get-Help about_Hash_Tables。


INPUTOBJECT：指定需要更新其屬性的 <IMigrationItem> 複製伺服器。 伺服器物件可以使用 Cmdlet Get-AzMigrateServerReplication取。
  - `[Location <String>]`：資源位置
  - `[CurrentJobId <String>]`：正在執行之工作的 ARM 識別碼。
  - `[CurrentJobName <String>]`：工作名稱。
  - `[CurrentJobStartTime <DateTime?>]`：工作的開始時間。
  - `[ProviderSpecificDetail <IMigrationProviderSpecificSettings>]`：移移提供者自訂設定。

NICTOUPDATE <IVCbtNicInput[]>：更新要建立 Azure VM 的 NIC。
  - `IsPrimaryNic <String>`：指出這是否為主要 NIC 的值。
  - `NicId <String>`：NIC 識別碼。
  - `[IsSelectedForMigration <String>]`：指出是否已選取此 NIC 進行移移的值。
  - `[TargetStaticIPAddress <String>]`：靜態 IP 位址。
  - `[TargetSubnetName <String>]`：目標子網名稱。
