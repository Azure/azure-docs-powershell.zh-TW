# Set-AzCdnEndpoint

## 簡介
更新 CDN 端點。

## 語法

```
Set-AzCdnEndpoint -CdnEndpoint <PSEndpoint> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 描述
**Set-AzCdnEndpoint** Cmdlet 會更新 AZURE 內容傳遞網路 (CDN) 端點。
