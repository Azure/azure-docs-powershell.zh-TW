# Get-AzCdnProfile

## 簡介
獲得 CDN 設定檔。

## 語法

```
Get-AzCdnProfile [-ProfileName <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 描述
**Get-AzCdnProfile** Cmdlet 會取得 Azure 內容傳遞網路 (CDN) 設定檔及其相關資訊。
