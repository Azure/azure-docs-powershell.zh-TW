# Remove-AzSqlServer

## 簡介
移除 Azure SQL Database 伺服器。

## 語法

```
Remove-AzSqlServer [-ServerName] <String> [-Force] [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
**Remove-AzSqlServer** Cmdlet 會移除 Azure SQL Database 伺服器。
刪除作業是非同步，可能需要一些時間，因此在執行任何取決於伺服器完全刪除的額外作業之前，請確認作業已完成。
例如，您需要建立使用相同名稱的新伺服器。

## 例子

### 範例 1：移除伺服器
```
PS C:\>Remove-AzSqlServer -ResourceGroupName "ResourceGroup01" -ServerName "Server01"
```

此命令會移除名為 Server01 的 Azure SQL Database 伺服器。
