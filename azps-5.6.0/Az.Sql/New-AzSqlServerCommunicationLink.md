# New-AzSqlServerCommunicationLink

## 簡介
為兩個 SQL Database 伺服器之間的彈性資料庫交易建立通訊連結。

## 語法

```
New-AzSqlServerCommunicationLink -LinkName <String> -PartnerServer <String> [-AsJob] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 描述
**New-AzSqlServerCommunicationLink** Cmdlet 會建立通訊連結，用於 Azure SQL Database 中兩個邏輯伺服器之間的固定資料庫交易。
量值資料庫交易可以橫跨其中一個配對伺服器中的資料庫。
您可以在伺服器上建立多個連結。
因此，彈性資料庫交易可能會橫跨許多伺服器。

## 例子

### 範例 1：建立通訊連結
```
PS C:\>New-AzSqlServerCommunicationLink -ResourceGroupName "ResourceGroup01" -ServerName "ContosoServer17" -LinkName "Link01" -PartnerServer "ContosoServer02"
```

此命令在 ContosoServer17 和 ContosoServer02 之間建立一個名為 Link01 的連結。
