# Get-AzSqlDatabaseDataMaskingRule

## 簡介
從資料庫獲得資料遮罩規則。

## 語法

```
Get-AzSqlDatabaseDataMaskingRule [-SchemaName <String>] [-TableName <String>] [-ColumnName <String>]
 [-ServerName] <String> [-DatabaseName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
**Get-AzSqlDatabaseDataMaskingRule** Cmdlet 會取得 Azure SQL 資料庫的特定資料遮罩規則或所有資料遮罩規則。
若要使用 Cmdlet，請使用 *ResourceGroupName、ServerName* 和 *DatabaseName* 參數來識別資料庫，並使用 *RuleId* 參數指定此 Cmdlet 會返回的規則。 
如果您沒有提供 *RuleId，* 會針對該 Azure SQL 資料庫的所有資料遮罩規則進行退貨。
Azure 上的 SQL Server 延伸資料庫服務也支援此 Cmdlet。
