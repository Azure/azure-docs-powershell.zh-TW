# Clear-AzSqlServerAdvancedThreatProtectionSetting

## 簡介
從伺服器移除進位威脅防護設定。

## 語法

```
Clear-AzSqlServerAdvancedThreatProtectionSetting [-PassThru] -ServerName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
此Clear-AzSqlServerAdvancedThreatProtectionSetting Cmdlet 會從 Azure SQL 伺服器移除進位威脅防護設定。
若要使用此 Cmdlet，請指定 ResourceGroupName 和 ServerName 參數，以識別此 Cmdlet 會移除設定的伺服器。

## 例子

### 範例 1：移除資料庫的進一步威脅防護設定
```
PS C:\> Clear-AzSqlServerAdvancedThreatProtectionSetting -ResourceGroupName "ResourceGroup11" -ServerName "Server01"
```

此命令會從伺服器名稱為 Server01 移除進位威脅防護設定。
