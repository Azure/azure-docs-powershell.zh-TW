# Get-AzSqlSyncAgent

## 簡介
會返回有關 Azure SQL 同步處理代理程式的資訊。

## 語法

```
Get-AzSqlSyncAgent [[-Name] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 描述
**Get-AzSqlSyncAgent Cmdlet** 會返回一或多個 Azure SQL 同步處理代理程式相關資訊。
指定同步代理程式的名稱，只查看該同步代理程式的資訊。
