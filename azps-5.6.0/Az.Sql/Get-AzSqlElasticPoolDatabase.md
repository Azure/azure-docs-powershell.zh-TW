# Get-AzSqlElasticPoolDatabase

## 簡介
在彈性資料庫及其屬性值中，獲得彈性資料庫。

## 語法

```
Get-AzSqlElasticPoolDatabase [-ElasticPoolName] <String> [-DatabaseName <String>] [-ServerName] <String>
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 描述
**Get-AzSqlElasticPoolDatabase** Cmdlet 會取得一個壓縮資料庫及其屬性值中的資料庫。
您可以在 Azure SQL Database 中指定彈性資料庫的名稱，只查看該資料庫的屬性值。

## 例子

### 範例 1：取得資料庫集中的所有資料庫
```
PS C:\> Get-AzSqlElasticPoolDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01"
```

此命令會獲得位於一個名為Pool01 的集中資料庫。

### 範例 2：使用篩選取得資料庫集中的所有資料庫
```
PS C:\> Get-AzSqlElasticPoolDatabase -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -ElasticPoolName "ElasticPool01" -DatabaseName "Database*"
```

此命令會獲得名稱為「資料庫」之一個名為「Pool01」的集中資料庫。
