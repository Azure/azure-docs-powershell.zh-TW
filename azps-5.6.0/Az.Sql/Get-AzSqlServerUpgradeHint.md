# Get-AzSqlServerUpgradeHint

## 簡介
獲得升級 Azure SQL Database 伺服器的價格層級提示。

## 語法

```
Get-AzSqlServerUpgradeHint [-ServerName] <String> [-ExcludeElasticPools <Boolean>]
 [-ResourceGroupName] <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 描述
**Get-AzSqlServerUpgradeHint** Cmdlet 會取得升級 Azure SQL Database 伺服器的價格層級提示。
提示可能包含彈性資料庫和獨立資料庫提示。
仍在 Web 和商務定價層級的資料庫會獲得升級至新基本、標準或進一步定價層級的提示，或進入彈性資料庫資料庫。
此 Cmdlet 會針對特定伺服器上託管的所有資料庫，會返回提示。

## 例子

### 範例 1：取得合併建議
```
PS C:\>Get-AzSqlServerUpgradeHint -ResourceGroupName "ResourceGroup01" -ServerName "Server01"
ElasticPools Databases           
------------ ---------           
{}           {database01, database02}
```

此命令會針對伺服器名稱為 Server01 的所有資料庫提供合併建議。
