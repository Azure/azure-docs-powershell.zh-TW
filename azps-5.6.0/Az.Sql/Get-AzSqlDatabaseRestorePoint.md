# Get-AzSqlDatabaseRestorePoint

## 簡介
會取回可還原 SQL 資料倉儲的明顯還原點。

## 語法

```
Get-AzSqlDatabaseRestorePoint [-ServerName] <String> [-DatabaseName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
**Get-AzSqlDatabaseRestorePoint** Cmdlet 會從 Azure SQL Data Warehouse 中取得可還原的還原點。
對於 Azure SQL Database，還原視窗是連續的。
這表示資料庫備份保留期間的任何時間點都可以做為還原點。
Azure 上的 SQL Server 延伸資料庫服務也支援此 Cmdlet。

## 例子

### 範例 1：取得所有還原點
```
PS C:\>Get-AzSqlDatabaseRestorePoint -ResourceGroupName "ResourceGroup01" -ServerName "Server01" -DatabaseName "Database01"
ResourceGroupName        : resourcegroup01
ServerName               : server01
DatabaseName             : database01
Location                 : Central US
RestorePointType         : CONTINUOUS
RestorePointCreationDate : 
EarliestRestoreDate      : 8/12/2015 12:00:00 AM
RestorePointLabel        : RestorePoint01
```

此命令會針對名為 Database01 的 Azure SQL Database，會返回所有可用的還原點。
