# Az.StackHCI 模組
## 描述
Microsoft Azure PowerShell：Azure Stack HCI 註冊 Cmdlet

## Az.StackHCI Cmdlet
### [Register-AzStackHCI](Register-AzStackHCI.md)
Register-AzStackHCI建立代表內部部署組群的 Microsoft.AzureStackHCI 雲端資源，然後向 Azure 登錄內部部署組。

### [Test-AzStackHCIConnection](Test-AzStackHCIConnection.md)
Test-AzStackHCIConnection驗證從內部部署組群節點到 Azure Stack HCI 所需 Azure 服務的連接。

### [取消註冊-AzStackHCI](Unregister-AzStackHCI.md)
Unregister-AzStackHCI Microsoft.AzureStackHCI 雲端資源代表內部部署組，並取消註冊 Azure 內部部署組。
如果未傳遞任何參數，該組組上可用的登錄資訊會用來取消註冊該組。
