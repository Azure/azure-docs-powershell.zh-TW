# Get-AzAppServicePlan

## 簡介
在指定的資源群組中，獲得 Azure 應用程式服務計畫。

## 語法

### S1
```
Get-AzAppServicePlan [[-ResourceGroupName] <String>] [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### S2
```
Get-AzAppServicePlan [-Location] <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 描述
**Get-AzAppServicePlan** Cmdlet 會取得指定資源群組中的 Azure 應用程式服務計畫。

## 例子

### 範例 1：從資源群組取得應用程式服務計畫
```
PS C:\>Get-AzAppServicePlan -ResourceGroupName "Default-Web-WestUS" -Name "ContosoASP"
```

此命令會獲得名為 ContosoASP 的應用程式服務計畫，該計畫屬於 Default-Web-WestUS 資源群組。

### 範例 2：在位置中取得所有應用程式服務方案
```
PS C:\>Get-AzAppServicePlan -Location "West US"
```

此命令會獲得位於「美國西部」地區的所有應用程式服務方案。
