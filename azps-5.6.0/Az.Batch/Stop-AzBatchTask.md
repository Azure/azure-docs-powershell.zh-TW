# Stop-AzBatchTask

## 簡介
停止批次處理工作。

## 語法

### Id
```
Stop-AzBatchTask [-JobId] <String> [-Id] <String> -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### InputObject
```
Stop-AzBatchTask [-Task] <PSCloudTask> -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 描述
**Stop-AzBatchTask** Cmdlet 會停止 Azure 批次處理工作。

## 例子

### 範例 1：刪除批次工作
```
PS C:\>Stop-AzBatchTask -JobId "Job-000001" -Id "Task23" -BatchContext $Context
```

此命令會停止在具有識別碼 Job-000001 的工作下具有識別碼工作 23 的工作。
命令會提示您確認。
使用 Get-AzBatchAccountKey Cmdlet 將上下文指派給$CoNtext變數。

### 範例 2：使用管線停止批次處理工作
```
PS C:\>Get-AzBatchTask -JobId "Job-000001" -Id "Task26" -BatchContext $Context | Stop-AzBatchTask -BatchContext $Context
```

此命令會使用識別碼 Job-000001，在具有識別碼 Job-000001 的Get-AzBatchTask工作。
該命令會使用管線運算子，將工作傳遞至目前的 Cmdlet。
命令會停止該工作。
