# Remove-AzBatchTask

## 簡介
刪除批次工作。

## 語法

### Id
```
Remove-AzBatchTask [-JobId] <String> [-Id] <String> [-Force] -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### InputObject
```
Remove-AzBatchTask [-InputObject] <PSCloudTask> [-Force] -BatchContext <BatchAccountContext>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
**Remove-AzBatchTask** Cmdlet 會刪除 Azure 批次處理工作。
除非您指定 Force 參數，否則此 Cmdlet 會提示您確認。

## 例子

### 範例 1：刪除批次工作
```
PS C:\>Remove-AzBatchTask -JobId "Job-000001" -Id "Task23" -BatchContext $Context
```

此命令會刪除識別碼為 Job-000001 的工作下具有識別碼工作 23 的工作。
命令會提示您確認。
使用 **Get-AzBatchAccountKey** Cmdlet 將上下文指派給$CoNtext變數。

### 範例 2：使用管線刪除批次工作而不確認
```
PS C:\>Get-AzBatchTask -JobId "Job-000001" -Id "Task26" -BatchContext $Context | Remove-AzBatchTask -Force -BatchContext $Context
```

此命令會使用 **Get-AzBatchTask Cmdlet** 在具有識別碼 Job-000001 的工作中取得識別碼工作 26 的批次工作。
該命令會使用管線運算子，將工作傳遞至目前的 Cmdlet。
命令會刪除該工作。
此命令會指定 *Force* 參數。
因此，命令不會提示您確認。
