# Update-AzTimeSeriesInsightsEnvironment

## 簡介
以指定的訂閱和資源群組中的指定名稱更新環境。

## 語法

### UpdateExpanded (預設) 
```
Update-AzTimeSeriesInsightsEnvironment -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-Tag <Hashtable>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### UpdateViaIdentityExpanded
```
Update-AzTimeSeriesInsightsEnvironment -InputObject <ITimeSeriesInsightsIdentity> [-Tag <Hashtable>]
 [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## 描述
以指定的訂閱和資源群組中的指定名稱更新環境。

## 筆記

別名

複雜的參數屬性

若要建立下列描述的參數，請建構包含適當屬性的雜湊表。 有關雜湊表的資訊，請執行Get-Help about_Hash_Tables。


INPUTOBJECT： <ITimeSeriesInsightsIdentity> 身分識別參數
  - `[AccessPolicyName <String>]`：存取策略的名稱。
  - `[EnvironmentName <String>]`：環境名稱
  - `[EventSourceName <String>]`：與指定環境相關聯的時間序列深入資訊事件來源名稱。
  - `[Id <String>]`：資源識別路徑
  - `[ReferenceDataSetName <String>]`：參照資料集的名稱。
  - `[ResourceGroupName <String>]`：Azure 資源組的名稱。
  - `[SubscriptionId <String>]`：Azure 訂閱識別碼。
