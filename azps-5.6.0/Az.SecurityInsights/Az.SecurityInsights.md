# Az.SecurityInsights 模組
## 描述
Microsoft Azure Sentinel 是一款可縮放、雲端原生的安全資訊事件管理 (SIEM) ，且安全性自動化回應 (SOAR) 解決方案。 Azure Sentinel 為整個企業提供智慧型安全性分析和威脅智慧，提供單一解決方案，用於警示偵測、威脅可見度、主動搜尋及威脅回應。
