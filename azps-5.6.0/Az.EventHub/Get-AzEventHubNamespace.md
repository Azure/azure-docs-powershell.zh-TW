# Get-AzEventHubNamespace

## 簡介
獲得事件中樞命名空間的詳細資訊，或獲得目前 Azure 訂閱中所有事件中樞命名空間的清單。

## 語法

```
Get-AzEventHubNamespace [[-ResourceGroupName] <String>] [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 描述
此Get-AzEventHubNamespace Cmdlet 會獲得指定事件中樞命名空間的詳細資訊，或目前 Azure 訂閱中所有事件中樞命名空間的清單。
如果提供命名空間名稱，則會返回單一事件中樞命名空間的詳細資訊。
如果未提供命名空間名稱，會返回命名空間清單。

## 例子

### 範例 1
```
PS C:\> Get-AzEventHubNamespace -ResourceGroupName MyResourceGroupName -NamespaceName MyNamespaceName

Tags                   :
ProvisioningState      : Succeeded
Status                 : Active

UpdatedAt              : 5/24/2019 12:48:14 AM
ServiceBusEndpoint     : #########
Enabled                : True
KafkaEnabled           : True
IsAutoInflateEnabled   : True
MaximumThroughputUnits : 10
```

在資源群組 \` \` MyResourceGroupName 中，獲得 \` 命名空間 MyNamespaceName 的詳細資訊 \` 。
