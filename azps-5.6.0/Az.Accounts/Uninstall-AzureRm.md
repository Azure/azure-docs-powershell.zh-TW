# Uninstall-AzureRm

## 簡介
從電腦移除所有 AzureRm 模組。

## 語法

```
Uninstall-AzureRm [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 描述
從電腦移除所有 AzureRm 模組。

## 例子

### 範例 1
```
PS C:\> Uninstall-AzureRm
```

執行此命令會針對執行 Cmdlet 的 PowerShell 版本，從電腦移除所有 AzureRm 模組。
