# Register-AzModule

## 簡介
僅適用于內部使用 - 針對 AutoLet 產生的 Cmdlet 提供執行時間支援

## 語法

```
Register-AzModule [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
僅適用于內部使用 - 針對 AutoLet 產生的 Cmdlet 提供執行時間支援

## 例子

### 範例 1
```powershell
PS C:\> Register-AzModule
```

自動刪除產生的 Cmdlet 在內部使用
