# Get-AzPrivateDnsVirtualNetworkLink

## 簡介
與指定的私人 DNS 區域相關聯的虛擬網路連結。

## 語法

```
Get-AzPrivateDnsVirtualNetworkLink -ResourceGroupName <String> -ZoneName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 描述
**Get-AzPrivateDnsVirtualNetworkLink** Cmdlet 會從指定的資源群組取得與特定私人 DNS 區域相關聯的虛擬網路連結。
如果您指定 *Name* 參數，會返回單 **一 PSPrivateDnsVirtualNetworkLink** 物件。
如果您未指定 *Name* 參數，會回到包含與指定資源群組中區域相關聯之所有連結的陣列。
您可以使用 **PSPrivateDnsVirtualNetworkLink** 物件來更新連結。
