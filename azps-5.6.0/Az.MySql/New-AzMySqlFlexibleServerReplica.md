# New-AzMySqlFlexibleServerReplica

## 簡介
為 MySQL 彈性伺服器建立複本伺服器

## 語法

```
New-AzMySqlFlexibleServerReplica -Replica <String> -ResourceGroupName <String> -Master <IServerAutoGenerated>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-AsJob] [-NoWait] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

## 描述
從現有資料庫建立新複本。

## 例子

### 範例 1：建立新 MySql 伺服器複本
```powershell
PS C:\> Get-AzMySqlFlexibleServer -ResourceGroupName PowershellMySqlTest -ServerName mysql-test | New-AzMySqlFlexibleServerReplica -Replica mysql-test-replica -ResourceGroupName PowershellMySqlTest

Name               Location AdministratorLogin Version StorageProfileStorageMb SkuName       SkuTier        
----               -------- ------------------ ------- ----------------------- -------       -------        
mysql-test-replica westus2   mysql_test         5.7     10240                   Standard_B1ms Burstable
```

此 Cmdlet 會建立新的 MySql 伺服器複本。

### 範例 2：建立新 MySql 伺服器複本
```powershell
PS C:\> $mysql = Get-AzMySqlFlexibleServer -ResourceGroupName PowershellMySqlTest -ServerName mysql-test
PS C:\> New-AzMySqlFlexibleServerReplica -Master $mysql -Replica mysql-test-replica -ResourceGroupName PowershellMySqlTest

Name               Location AdministratorLogin Version StorageProfileStorageMb SkuName       SkuTier        
----               -------- ------------------ ------- ----------------------- -------       -------        
mysql-test-replica westus2   mysql_test         5.7     10240                   Standard_B1ms Burstable
```

此 Cmdlet 具有參數主控 (inputobject) 建立新的 MySql 伺服器複本。

## 筆記

別名

複雜的參數屬性

若要建立下列描述的參數，請建構包含適當屬性的雜湊表。 有關雜湊表的資訊，請執行Get-Help about_Hash_Tables。


<IServerAutoGenerated>MASTER：建立複本的來源伺服器物件。
  - `Location <String>`：資源所在地的地理位置
  - `[Tag <ITrackedResourceTags>]`：資源標記。
    - `[(Any) <String>]`：這表示任何屬性都可以新增到這個物件。
  - `[AdministratorLogin <String>]`：系統管理員的伺服器登入名稱。 只能在建立伺服器時指定， (建立時) 。
  - `[AdministratorLoginPassword <SecureString>]`：系統管理員登入密碼 (建立伺服器時) 。
  - `[AvailabilityZone <String>]`：伺服器的可用性區域資訊。
  - `[CreateMode <CreateMode?>]`：建立新 MySQL 伺服器的模式。
  - `[DelegatedSubnetArgumentSubnetArmResourceId <String>]`：委派子網 arm 資源識別碼。
  - `[HaEnabled <HaEnabledEnum?>]`：為伺服器啟用或不啟用 HA。
  - `[IdentityType <ResourceIdentityType?>]`：身分識別類型。
  - `[InfrastructureEncryption <InfrastructureEncryptionEnum?>]`：狀態顯示伺服器是否啟用基礎結構加密。
  - `[MaintenanceWindowCustomWindow <String>]`：表示自訂視窗是啟用或停用
  - `[MaintenanceWindowDayOfWeek <Int32?>]`：維護視窗的一周中的日
  - `[MaintenanceWindowStartHour <Int32?>]`：維護視窗的開始時間
  - `[MaintenanceWindowStartMinute <Int32?>]`：維護視窗的開始時間
  - `[PropertiesTag <IServerPropertiesTags>]`：金鑰值組形式的應用程式特定中繼資料。
    - `[(Any) <String>]`：這表示任何屬性都可以新增到這個物件。
  - `[ReplicationRole <String>]`：複製角色。
  - `[RestorePointInTime <DateTime?>]`：使用 ISO8601 (還原點建立) ，指定還原的時間。
  - `[SkuName <String>]`：SKU 的名稱，例如Standard_D32s_v3。
  - `[SkuTier <SkuTier?>]`：特定 SKU 的層級，例如 GeneralPurpose。
  - `[SourceServerId <String>]`：來源 MySQL 伺服器識別碼。
  - `[SslEnforcement <SslEnforcementEnum?>]`：在連接至伺服器時啟用 ssl 強制執行或不啟用。
  - `[StorageProfileBackupRetentionDay <Int32?>]`：伺服器的備份保留天數。
  - `[StorageProfileStorageAutogrow <StorageAutogrow?>]`：啟用儲存空間自動放大。
  - `[StorageProfileStorageIop <Int32?>]`：伺服器儲存空間 IOPS。
  - `[StorageProfileStorageMb <Int32?>]`：伺服器允許的最大儲存空間。
  - `[Version <ServerVersion?>]`：伺服器版本。
