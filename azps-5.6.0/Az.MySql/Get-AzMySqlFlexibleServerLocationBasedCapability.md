# Get-AzMySqlFlexibleServerLocationBasedCapability

## 簡介
取得位置的可用 SKU 資訊

## 語法

```
Get-AzMySqlFlexibleServerLocationBasedCapability -Location <String> [-SubscriptionId <String[]>]
 [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## 描述
取得位置的可用 SKU 資訊

## 例子

### 範例 1：根據位置名稱取得位置功能
