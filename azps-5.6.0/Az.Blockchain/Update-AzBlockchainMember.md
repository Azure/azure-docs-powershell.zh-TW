# Update-AzBlockchainMember

## 簡介
更新網路管理成員。

## 語法

### UpdateExpanded (預設) 
```
Update-AzBlockchainMember -Name <String> -ResourceGroupName <String> [-SubscriptionId <String>]
 [-ConsortiumManagementAccountPassword <SecureString>] [-FirewallRule <IFirewallRule[]>]
 [-Password <SecureString>] [-Tag <Hashtable>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

### UpdateViaIdentityExpanded
```
Update-AzBlockchainMember -InputObject <IBlockchainIdentity>
 [-ConsortiumManagementAccountPassword <SecureString>] [-FirewallRule <IFirewallRule[]>]
 [-Password <SecureString>] [-Tag <Hashtable>] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf]
 [<CommonParameters>]
```

## 描述
更新網路管理成員。

## 例子

### 範例 1：更新成員
```powershell
PS C:\> $passwd2 = 'strongMemberAccountPassword@2' | ConvertTo-SecureString -AsPlainText -Force
PS C:\> Update-AzBlockchainMember -Name dolauli002 -ResourceGroupName testgroup -Password $passwd2

Location Name       Type
-------- ----       ----
eastus   dolauli002 Microsoft.Blockchain/blockchainMembers
```

此命令會更新更新成員。

### 範例 2：更新成員
```powershell
PS C:\> $tag = @{'againupdate'='password'}
PS C:\> $member = Get-AzBlockchainMember -Name $env.blockchainMember -ResourceGroupName $env.resourceGroup
PS C:\> Update-AzBlockchainMember -InputObject $member -Tag $tag

Location Name       Type
-------- ----       ----
eastus   dolauli002 Microsoft.Blockchain/blockchainMembers
```

此命令會更新更新成員。

## 筆記

別名

複雜的參數屬性

若要建立下列描述的參數，請建構包含適當屬性的雜湊表。 有關雜湊表的資訊，請執行Get-Help about_Hash_Tables。


FIREWALLRULE <IFirewallRule[]>：獲取或設定防火牆規則。
  - `[EndIPAddress <String>]`：獲取或設定防火牆規則範圍的結束 IP 位址。
  - `[RuleName <String>]`：獲取或設定防火牆規則的名稱。
  - `[StartIPAddress <String>]`：獲取或設定防火牆規則範圍的起始 IP 位址。

INPUTOBJECT： <IBlockchainIdentity> 身分識別參數
  - `[BlockchainMemberName <String>]`：進位會員名稱。
  - `[Id <String>]`：資源識別路徑
  - `[Location <String>]`：位置名稱。
  - `[OperationId <String>]`：操作識別碼。
  - `[ResourceGroupName <String>]`：包含資源的資源組名。 您可以從 Azure Resource Manager API 或入口網站取得此值。
  - `[SubscriptionId <String>]`：獲得可唯一識別 Microsoft Azure 訂閱的訂閱識別碼。 訂閱識別碼是每個服務通話 URI 的一部分。
  - `[TransactionNodeName <String>]`：交易處理節點名稱。
