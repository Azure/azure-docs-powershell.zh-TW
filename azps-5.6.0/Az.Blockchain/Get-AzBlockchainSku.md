# Get-AzBlockchainSku

## 簡介
列出資源類型的 SKU。

## 語法

```
Get-AzBlockchainSku [-SubscriptionId <String[]>] [-DefaultProfile <PSObject>] [<CommonParameters>]
```

## 描述
列出資源類型的 SKU。

## 例子

### 範例 1：訂閱的清單 SKU
```powershell
PS C:\> Get-AzBlockchainSku -SubscriptionId c9cbd920-c00c-427c-852b-8aaf38badaeb

```

此命令會列出訂閱的 SKU。
