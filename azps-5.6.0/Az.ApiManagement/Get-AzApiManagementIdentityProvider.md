# Get-AzApiManagementIdentityProvider

## 簡介
取得身分識別提供者組組詳細資料。

## 語法

### AllIdentityProviders (預設) 
```
Get-AzApiManagementIdentityProvider -Context <PsApiManagementContext>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### IdentityProviderByType
```
Get-AzApiManagementIdentityProvider -Context <PsApiManagementContext>
 -Type <PsApiManagementIdentityProviderType> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 描述
取得身分識別提供者組組詳細資料。
ClientSecret 不會包含在結果詳細資料中。 若要取得用戶端密碼，請使用 **Get-AzApiManagementIdentityProviderClientSecret。**

## 例子

### 範例 1：取得所有身分識別提供者

```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\>Get-AzApiManagementIdentityProvider -Context $apimContext
```

取得服務上的所有身分識別提供者組配置。

### 範例 2：取得 AAD 類型身分識別提供者
```powershell
PS C:\>$apimContext = New-AzApiManagementContext -ResourceGroupName "Api-Default-WestUS" -ServiceName "contoso"
PS C:\> Get-AzApiManagementIdentityProvider -Context $apimContext -Type Aad

ResourceGroupName        : Api-Default-West-US
ServiceName              : contoso
```

獲得 Azure Active Directory 的身分識別提供者組配置。
