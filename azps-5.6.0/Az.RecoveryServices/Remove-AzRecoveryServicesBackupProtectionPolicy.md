# Remove-AzRecoveryServicesBackupProtectionPolicy

## 簡介
從儲存庫刪除備份保護原則。

## 語法

### PolicyName (預設) 
```
Remove-AzRecoveryServicesBackupProtectionPolicy [-Name] <String> [-PassThru] [-Force] [-VaultId <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### PolicyObject
```
Remove-AzRecoveryServicesBackupProtectionPolicy [-Policy] <PolicyBase> [-PassThru] [-Force] [-VaultId <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
**Remove-AzRecoveryServicesBackupProtectionPolicy** Cmdlet 會刪除儲存庫的備份策略。
在您刪除備份保護原則之前，該策略不得有任何相關聯的備份專案。
刪除該策略之前，請確定每個相關聯的專案都與一些其他策略相關聯。
若要將另一個策略與備份專案關聯，請使用 Enable-AzRecoveryServicesBackupProtection Cmdlet。
使用目前的 Cmdlet 之前，使用 Set-AzRecoveryServicesVaultContext Cmdlet 來設定庫內容。

## 例子

### 範例 1：移除策略
```powershell
PS C:\>$Pol= Get-AzRecoveryServicesBackupProtectionPolicy -Name "NewPolicy"
PS C:\> Remove-AzRecoveryServicesBackupProtectionPolicy -Policy $Pol
```

第一個命令會獲得名為 NewPolicy 的備份保護原則，然後將它儲存在 $Pol 變數中。
第二個命令會移除 $Pol 中$Pol。

### 範例 2

從儲存庫刪除備份保護原則。  (自動) 

```powershell <!-- Aladdin Generated Example --> 
Remove-AzRecoveryServicesBackupProtectionPolicy -Name 'V2VM' -VaultId $vault.ID
```
