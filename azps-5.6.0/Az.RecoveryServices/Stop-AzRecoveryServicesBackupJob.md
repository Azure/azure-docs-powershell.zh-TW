# Stop-AzRecoveryServicesBackupJob

## 簡介
取消進行中的工作。

## 語法

### JobFilterSet (預設) 
```
Stop-AzRecoveryServicesBackupJob [-Job] <JobBase> [-VaultId <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### IdFilterSet
```
Stop-AzRecoveryServicesBackupJob [-JobId] <String> [-VaultId <String>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
**Stop-AzRecoveryServicesBackupJob** Cmdlet 會取消現有的 Azure 備份工作。
使用此 Cmdlet 可以停止花費太多時間並阻止其他活動的工作。
您可以只取消備份和還原工作類型。
使用目前的 Cmdlet 之前，使用 Set-AzRecoveryServicesVaultContext Cmdlet 來設定庫內容。

## 例子

### 範例 1：停止備份工作
```
PS C:\>$Job = Get-AzRecoveryServicesBackupJob -Operation Backup
PS C:\> Stop-AzRecoveryServicesBackupJob -JobID $Job.InstanceId
```

第一個命令會獲得備份工作，然後將該工作儲存在$Job變數。
最後一個命令會停止工作，在 $Job 中指定備份工作的實例識別碼。
