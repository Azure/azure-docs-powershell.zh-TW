# Get-AzLogicAppUpgradedDefinition

## 簡介
獲得邏輯應用程式的升級定義。

## 語法

```
Get-AzLogicAppUpgradedDefinition -ResourceGroupName <String> -Name <String> -TargetSchemaVersion <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 描述
**Get-AzLogicAppUpgradedDefinition Cmdlet** 會從資源群組取得架構版本和邏輯應用程式的升級定義。
此 Cmdlet 會返回代表升級邏輯應用程式定義的物件。
指定資源組名、邏輯應用程式名稱和目標架構版本。
此模組支援動態參數。
若要使用動態參數，請于命令中輸入。
若要探索動態參數的名稱，在 Cmdlet 名稱之後輸入連字號 (-) ，然後重複按 Tab 鍵以迴圈流覽可用的參數。
如果您省略必要的範本參數，Cmdlet 會提示您輸入值。
