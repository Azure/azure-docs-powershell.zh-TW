# Stop-AzLogicAppRun

## 簡介
取消邏輯應用程式的執行。

## 語法

```
Stop-AzLogicAppRun -ResourceGroupName <String> -Name <String> -RunName <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
**Stop-AzLogicAppRun** Cmdlet 會取消邏輯應用程式的執行。
指定邏輯應用程式、資源群組及執行。
此模組支援動態參數。
若要使用動態參數，請于命令中輸入。
若要探索動態參數的名稱，在 Cmdlet 名稱之後輸入連字號 (-) ，然後重複按 Tab 鍵以迴圈流覽可用的參數。
如果您省略必要的範本參數，Cmdlet 會提示您輸入值。

## 例子

### 範例 1：取消執行邏輯應用程式
```
PS C:\>Stop-AzLogicAppRun -ResourceGroupName "ResourceGroup11" -Name "LogicApp03" -RunName "08587489104702792076" -Force
```

此命令會取消邏輯應用程式 LogicApp03 的執行。
