# Get-AzIntegrationAccountAgreement

## 簡介
獲得整合帳戶協定。

## 語法

```
Get-AzIntegrationAccountAgreement [-ResourceGroupName <String>] [-Name <String>] [-AgreementName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 描述
**Get-AzEmentAccountAgreement Cmdlet** 會從 Azure 資源群組取得整合帳戶協定。
指定整合帳戶名稱、資源組名和協定名稱。
此模組支援動態參數。
若要使用動態參數，請于命令中輸入。
若要探索動態參數的名稱，在 Cmdlet 名稱之後輸入連字號 (-) ，然後重複按 Tab 鍵以迴圈流覽可用的參數。
如果您省略必要的範本參數，Cmdlet 會提示您輸入值。
