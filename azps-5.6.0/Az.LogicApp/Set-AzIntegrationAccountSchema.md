# Set-AzIntegrationAccountSchema

## 簡介
修改整合帳戶架構。

## 語法

```
Set-AzIntegrationAccountSchema -ResourceGroupName <String> -Name <String> -SchemaName <String>
 [-SchemaFilePath <String>] [-SchemaDefinition <String>] [-SchemaType <String>] [-ContentType <String>]
 [-Metadata <Object>] [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 描述
**Set-AzCountAccountSchdlet** 會修改整合帳戶架構。
此 Cmdlet 會返回代表整合帳戶架構的物件。
指定整合帳戶名稱、資源組名和架構名稱。
在命令列中指定的範本參數檔案值優先于範本參數物件的範本參數值。
此模組支援動態參數。
若要使用動態參數，請于命令中輸入。
若要探索動態參數的名稱，在 Cmdlet 名稱之後輸入連字號 (-) ，然後重複按 Tab 鍵以迴圈流覽可用的參數。
如果您省略必要的範本參數，Cmdlet 會提示您輸入值。
