# Remove-AzIntegrationAccountReceivedIcn

## 簡介
此 Cmdlet 會移除每個協定的特定接收交換控制項編號及控制項編號值。

## 語法

```
Remove-AzIntegrationAccountReceivedIcn -ResourceGroupName <String> -Name <String> -AgreementName <String>
 -ControlNumberValue <String> [-AgreementType <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## 描述
此 Cmdlet 旨在用於災害復原案例，以從整合帳戶移除收到的交換控制編號，讓 B2B 連接器在啟用重複數位偵測時，可以再次處理郵件。
在少數情況下，接收的交換控制號碼可能在發生災害前和 B2B 連接器拒絕交換之前保留為錯誤。
在這類情況下，作業可能會想要讓修復網站在有效負載修正之後，再次處理相同的交換。
請提供 "-AgreementType" 參數，以指定要傳回 X12 或 Edifact 控制項編號
