# Get-AzIntegrationAccountCertificate

## 簡介
從資源群組中獲得整合帳戶憑證。

## 語法

```
Get-AzIntegrationAccountCertificate [-ResourceGroupName <String>] [-Name <String>] [-CertificateName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 描述
**Get-AzIficAccountCertificate** Cmdlet 會從資源群組取得整合帳戶憑證。
指定整合帳戶名稱、資源組名和憑證名稱。
此模組支援動態參數。
若要使用動態參數，請于命令中輸入。
若要探索動態參數的名稱，在 Cmdlet 名稱之後輸入連字號 (-) ，然後重複按 Tab 鍵以迴圈流覽可用的參數。
如果您省略必要的範本參數，Cmdlet 會提示您輸入值。
