# Stop-AzStreamAnalyticsJob

## 簡介
停止 Stream Analytics 工作。

## 語法

```
Stop-AzStreamAnalyticsJob [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 描述
**Stop-AzStreamAnalyticsJob** Cmdlet 會非同步停止 Stream Analytics 工作在 Azure 中執行，並解除配置之前使用的資源。
您可以透過 Azure 入口網站和管理 API，在訂閱中持續提供工作定義和中繼資料，如此一來，工作就可以編輯並重新啟動。
在已停止狀態的工作不會收取任何費用。

## 例子

### 範例 1：停止進行中的工作
```powershell
PS C:\>Stop-AzStreamAnalyticsJob -ResourceGroupName "StreamAnalytics-Default-West-US" -Name "StreamingJob"
```

此命令會停止 StreamingJob 的工作。
