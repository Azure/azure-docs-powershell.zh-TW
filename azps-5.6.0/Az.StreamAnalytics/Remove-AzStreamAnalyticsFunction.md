# Remove-AzStreamAnalyticsFunction

## 簡介
刪除 Stream Analytics 工作中的函數。

## 語法

```
Remove-AzStreamAnalyticsFunction [-JobName] <String> [-Name] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
**Remove-AzStreamAnalyticsFunctdion Cmdlet** 會非同步刪除 Azure Stream Analytics 工作中的函數。

## 例子

### 範例 1：移除 Stream Analytics 函數
```
PS C:\>Remove-AzStreamAnalyticsFunction -ResourceGroupName "StreamAnalytics-Default-West-US" -JobName "StreamJob22" -Name "ScoreTweet"
```

此命令會從名為 StreamJob22 的工作移除名為 ScoreTweet 的函數。
