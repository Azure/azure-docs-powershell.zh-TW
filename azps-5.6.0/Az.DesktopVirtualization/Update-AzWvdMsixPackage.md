# Update-AzWvdMsixPackage

## 簡介
更新 MSIX 套件。

## 語法

### UpdateExpanded (預設) 
```
Update-AzWvdMsixPackage -FullName <String> -HostPoolName <String> -ResourceGroupName <String>
 [-SubscriptionId <String>] [-DisplayName <String>] [-IsActive] [-IsRegularRegistration]
 [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### UpdateViaIdentityExpanded
```
Update-AzWvdMsixPackage -InputObject <IDesktopVirtualizationIdentity> [-DisplayName <String>] [-IsActive]
 [-IsRegularRegistration] [-DefaultProfile <PSObject>] [-Confirm] [-WhatIf] [<CommonParameters>]
```

## 描述
更新 MSIX 套件。

## 例子

### 範例 1：更新 MSIX 套件 
```powershell
PS C:\> Update-AzWvdMsixPackage -HostPoolName HostPoolName `
                -ResourceGroupName ResourceGroupName `
                -SubscriptionId SubscriptionId `
                -displayName 'Updated-display-Name' `
                    -IsRegularRegistration:$False `
                -IsActive:$True

Name                                                  Type
----                                                  ----
HostPoolName/MSIXPackage_FullName1                    Microsoft.DesktopVirtualization/hostpools/msixpackages
```

此命令會更新 HostPool 中的 MSIX 套件。

別名

複雜的參數屬性

若要建立下列描述的參數，請建構包含適當屬性的雜湊表。 有關雜湊表的資訊，請執行Get-Help about_Hash_Tables。


INPUTOBJECT： <IDesktopVirtualizationIdentity> 身分識別參數
  - `[ApplicationGroupName <String>]`：應用程式組的名稱
  - `[ApplicationName <String>]`：指定應用程式群組中的應用程式名稱
  - `[DesktopName <String>]`：指定桌面群組中的桌面名稱
  - `[HostPoolName <String>]`：指定資源群組內的主機組名
  - `[Id <String>]`：資源識別路徑
  - `[MsixPackageFullName <String>]`：指定主機多工緩衝處理程式中 MSIX 套件的版本特定套件完整名稱
  - `[ResourceGroupName <String>]`：資源組的名稱。 名稱不區分大小寫。
  - `[SessionHostName <String>]`：指定主機集中的工作階段主機名稱
  - `[SubscriptionId <String>]`：目標訂閱的識別碼。
  - `[UserSessionId <String>]`：指定工作階段主機內的使用者會話名稱
  - `[WorkspaceName <String>]`：工作區的名稱
