# Get-AzLegacyPeering

## 簡介
用來將舊版對等資源轉換為 Azure 資源管理 (ARM) 資源。 

## 語法

```
Get-AzLegacyPeering [-PeeringLocation] <String> [-Kind] <String> [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## 描述
該命令用來查看舊版對等資源，而您全部都將其轉換為 AZURE 資源管理 (ARM) 資源。
