# New-AzFrontDoorRulesEngineRuleObject

## 簡介
建立 PSRulesEngineRule 物件以建立規則引擎。

## 語法

```
New-AzFrontDoorRulesEngineRuleObject -Name <String> -Priority <Int32> -Action <PSRulesEngineAction>
 [-MatchProcessingBehavior <PSMatchProcessingBehavior>] [-MatchCondition <PSRulesEngineMatchCondition[]>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 描述
建立 PSRulesEngineRule 物件以建立規則引擎。

使用 Cmdlet "New-AzFrontDoorRulesEngineActionObject" 建立 PSRulesEngineAction 物件，以進入 "-Action" 參數。
使用 Cmdlet "New-AzFrontDoorRulesEngineMatchConditionObject" 來建立 PSRulesEngineMatchCondition 物件，以進入 "-MatchCondition" 參數。
