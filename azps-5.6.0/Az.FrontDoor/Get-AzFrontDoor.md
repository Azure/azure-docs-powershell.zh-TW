# Get-AzFrontDoor

## 簡介
取得前門負載平衡器

## 語法

```
Get-AzFrontDoor [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## 描述
**Get-AzFrontDoor** CmdletGet 會取得目前訂閱中所有符合所提供的資訊的現有前門。
