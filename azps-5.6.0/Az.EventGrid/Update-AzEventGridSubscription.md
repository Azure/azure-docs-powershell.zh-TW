## 描述
更新事件格線事件訂閱的屬性。 這可以用來更新現有事件訂閱的篩選、目的地或標籤。

## 例子

### 範例 1
```powershell
PS C:\> Update-AzEventGridSubscription -EventSubscriptionName ES1 -TopicName Topic1 -ResourceGroup MyResourceGroupName -Endpoint https://requestb.in/1kxxoui1
```

將資源群組 \` \` \` \` \` MyResourceGroupName 中主題 1 的事件訂閱 ES1 端點更新 \` 為 \`https://requestb.in/1kxxoui1\`

### 範例 2
```powershell
PS C:\> Get-AzEventGridSubscription -EventSubscriptionName ES1 -TopicName Topic1 -ResourceGroup MyResourceGroupName | Update-AzEventGridSubscription -Endpoint https://requestb.in/1kxxoui1
```

將資源群組 \` \` \` \` \` MyResourceGroupName 中主題 1 的事件訂閱 ES1 端點更新 \` 為 \`https://requestb.in/1kxxoui1\`

### 範例 3
```powershell
PS C:\> Update-AzEventGridSubscription -EventSubscriptionName ES1 -ResourceId "/subscriptions/55f3dcd4-cac7-43b4-990b-a139d62a1eb2/resourceGroups/TestRG/providers/Microsoft.EventHub/namespaces/ContosoNamespace" -Endpoint https://requestb.in/1kxxoui1 -SubjectEndsWith "jpg"
```

將 EventHub 命名空間 ContosoNamespace 的事件訂閱 ES1 屬性更新為 ' 的新端點，以及新的 \` \` \` https://requestb.in/1kxxoui1\ SubjectEndsWith 篩選為 \` jpg\`

### 範例 4
```powershell
PS C:\> $labels = "Finance", "HR"
PS C:\> Update-AzEventGridSubscription -EventSubscriptionName ES1 -ResourceGroup MyResourceGroupName -Label $labels
```

將資源群組 \` \` \` MyResourceGroupName 的事件訂閱 ES1 屬性更新為新的 \` $labels。

## 參數

### -AdvancedFilter
指定用於屬性型篩選之多個雜湊表值的陣列的進一級篩選。 每個雜湊表值都有下列索引鍵值資訊：運算、索引鍵和值或值。 運算子可以是下列其中一個值：NumberIn、NumberNotIn、NumberLessQual、NumberGreaterQual、NumberLessQualOrEquals、NumberGreaterQualOrEquals、BoolEquals、StringIn、StringNotIn、StringBeginsWith、StringEndsWith 或 StringContains。 Key 代表已使用進位篩選原則的負載屬性。 最後，值或值代表要相符的值或一組值。 這可以是對應類型的單一值或值陣列。 做為進一步篩選參數的範例：$AdvancedFilters=@ ($AdvFilter 1、$AdvFilter 2) 其中 $AdvFilter 1=@{operator="NumberIn"; key="Data.Key1";Values=@ (1，2) } 和 $AdvFilter 2=@{operator="StringBeginsWith"; key="Subject";Values=@ ("SubjectPrefix1"，"SubjectPrefix2") }
