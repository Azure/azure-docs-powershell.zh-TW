# Enable-AzStorageDeleteRetentionPolicy

## 簡介
啟用 Azure 儲存體 Blob 服務的刪除保留原則。

## 語法

```
Enable-AzStorageDeleteRetentionPolicy [-RetentionDays] <Int32> [-PassThru] [-Context <IStorageContext>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
**Enable-AzStorageDeleteRetentionPolicy** Cmdlet 可啟用 Azure 儲存體 Blob 服務的刪除保留原則。

## 例子

### 範例 1：啟用 Blob 服務的刪除保留原則
```
C:\PS>Enable-AzStorageDeleteRetentionPolicy -RetentionDays 3
```

此命令會啟用 Blob 服務的刪除保留原則，並且將已刪除的 Blob 保留天數設為 3。
