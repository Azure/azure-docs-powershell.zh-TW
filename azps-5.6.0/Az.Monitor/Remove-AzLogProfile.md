# Remove-AzLogProfile

## 簡介
移除記錄設定檔。

## 語法

```
Remove-AzLogProfile -Name <String> [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 描述
**Remove-AzLogProfile** Cmdlet 會移除記錄設定檔。
此 Cmdlet 實做 ShouldProcess 模式，即實際建立、修改或移除資源之前，可能會向使用者要求確認。

## 例子

### 範例 1

移除記錄設定檔。  (自動) 

```powershell <!-- Aladdin Generated Example --> 
Remove-AzLogProfile -Name 'LogAlertRule1'
```
