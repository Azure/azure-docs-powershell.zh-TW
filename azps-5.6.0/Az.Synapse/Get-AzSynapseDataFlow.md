# Get-AzSynapseDataFlow

## 簡介
在工作區中獲取資料流程相關資訊。

## 語法

### GetByName (預設) 
```
Get-AzSynapseDataFlow -WorkspaceName <String> [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

### GetByObject
```
Get-AzSynapseDataFlow -WorkspaceObject <PSSynapseWorkspace> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 描述
**Get-AzSynapseDataFlow** Cmdlet 會取得工作區中資料流程的資訊。
如果您指定資料流程的名稱，此 Cmdlet 會獲得該資料流程的資訊。
如果您未指定名稱，此 Cmdlet 會獲得工作區中所有資料流量的資訊。

## 例子

### 範例 1
```powershell
PS C:\> Get-AzSynapseDataFlow -WorkspaceName ContosoWorkspace
```

此命令會獲得名為 ContosoWorkspace 工作區中所有資料流程程的資訊。

### 範例 2
```powershell
PS C:\> Get-AzSynapseDataFlow -WorkspaceName ContosoWorkspace -Name ContosoDataFlow
```

此命令會從名為 ContosoWorkspace 的工作區中，獲得名為 ContosoDataFlow 的資料流程相關資訊。
