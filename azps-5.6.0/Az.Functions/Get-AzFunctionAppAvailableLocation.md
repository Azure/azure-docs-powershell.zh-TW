# Get-AzFunctionAppAvailableLocation

## 簡介
為指定作業系統和計畫類型提供函數應用程式的位置。

## 語法

```
Get-AzFunctionAppAvailableLocation [[-SubscriptionId] <String[]>] [[-PlanType] <String>] [[-OSType] <String>]
 [[-DefaultProfile] <PSObject>] [<CommonParameters>]
```

## 描述
為指定作業系統和計畫類型提供函數應用程式的位置。

## 例子
