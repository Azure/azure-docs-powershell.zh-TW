# ConvertTo-AzVMManagedDisk

## 簡介
將具有 Blob 型磁片的虛擬機器轉換為受管理磁片的虛擬機器。

## 語法

```
ConvertTo-AzVMManagedDisk [-ResourceGroupName] <String> [-VMName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
**ConvertTo-Az VIRTUALManagedObdlet Cmdlet** 會將具有 Blob 型磁片的虛擬機器轉換為具有受管理磁片的虛擬機器。
在重新執行此作業之前，必須先停止處理虛擬機器。

## 例子

### 範例 1
```
PS C:\> ConvertTo-AzVMManagedDisk -ResourceGroupName 'ResourceGroup01' -VMName 'VM01'
```

此命令會將資源群組 "ResourceGroup01" 中名為 "VM01" 的虛擬機器 Blob 型磁片轉換為受管理的磁片。
