# Remove-AzVMDscExtension

## 簡介
從資源群組中的虛擬機器移除 DSC 擴充處理常式。

## 語法

```
Remove-AzVMDscExtension [-ResourceGroupName] <String> [-VMName] <String> [[-Name] <String>] [-NoWait]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
**Remove-Az VIRTUALDscExtension** Cmdlet 會從資源群組中的虛擬機器移除想要的狀態組態 (DSC) 副檔名處理常式。

## 例子

### 範例 1：移除 DSC 擴充功能
```
PS C:\> Remove-AzVMDscExtension -ResourceGroupName "ResourceGroup001" -VMName "VM07" -Name "DSC"
```

此命令會移除名為 VM07 的虛擬機器上名為 DSC 的擴充功能。
