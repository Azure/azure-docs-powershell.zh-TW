# Get-AzSnapshot

## 簡介
獲得快照的屬性

## 語法

```
Get-AzSnapshot [[-ResourceGroupName] <String>] [[-SnapshotName] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 描述
**Get-AzSnapshot** Cmdlet 會取得快照的屬性。
