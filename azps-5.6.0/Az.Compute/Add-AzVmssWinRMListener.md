# Add-AzVmssWinRMListener

## 簡介
將 WinRM 聆聽者新增到 VMSS。

## 語法

```
Add-AzVmssWinRMListener [-VirtualMachineScaleSet] <PSVirtualMachineScaleSet> [[-Protocol] <ProtocolTypes>]
 [[-CertificateUrl] <String>] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 描述
**Add-Az VmssWinRMListener Cmdlet** 會新增 Windows 遠端系統管理 (WinRM) 在虛擬機器縮放集 (VMSS) 。

## 例子

### 範例 1：新增 WinRM 聆聽者至 VMSS
```
PS C:\> $VMSS = New-AzVmssConfig
PS C:\> Add-AzVmssWinRMListener -VirtualMachineScaleSet $VMSS -Protocol Https -CertificateUrl "http://keyVaultName.vault.contoso.net/secrets/secretName/secretVersion"
```

此範例會將 WinRM 聆聽者新增到 VMSS。
第一個命令使用 **New-Az VmssConfig** Cmdlet 來建立 VMSS 組式物件，並儲存在名為 $VMSS 的變數中。
第二個命令會將 HTTP 通訊協定 WinRM 聆聽者與憑證一起新增到 VMSS 的指定 URL。
