# Az.DataFactory 模組
## 描述
Azure Data Factory V2 是除了 Azure Data Factory V1 的更新和批次處理時間序列記錄資料之外的資料整合平臺，一般用途應用程式模型支援新式資料倉儲模式與案例、lift-and-shift SSIS，以及資料導向 SaaS 應用程式。 大規模撰寫及管理可靠且安全的資料整合工作流程。 使用原生 ADF 資料連線器和整合執行時間來移動及轉換雲端和內部部署資料，這些資料可以是非結構化、半結構化，而且具有 Hadoop、Azure Data Lake、Spark、SQL Server、Runtimes DB 和許多其他資料平臺。
