# Remove-AzDataFactoryHub

## 簡介
從 Azure Data Factory 移除中樞。

## 語法

### ByFactoryName (預設) 
```
Remove-AzDataFactoryHub [-Name] <String> [-Force] [-DataFactoryName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ByFactoryObject
```
Remove-AzDataFactoryHub [-Name] <String> [-Force] [-DataFactory] <PSDataFactory>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
**Remove-AzDataFactoryHub** Cmdlet 會從指定的 Azure 資源群組和指定資料工廠中的 Azure Data Factory 移除中樞。
如果您移除中樞，也會移除中樞中所有連結的服務和管線。

## 例子

### 範例 1：移除中樞
```
PS C:\>Remove-AzDataFactoryHub -ResourceGroupName "ADFResourceGroup" -DataFactoryName "ADFDataFactory" -Name "ContosoDataHub"
```

此命令會從名為 ADFResourceGroup 的 Azure 資源群組和名為 ADFDataFactory 的資料工廠移除名為 ContosoDataHub 的中樞。
