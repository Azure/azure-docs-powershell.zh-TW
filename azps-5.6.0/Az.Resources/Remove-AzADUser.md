# Remove-AzADUser

## 簡介
刪除 Active Directory 使用者。

## 語法

### UPNOrObjectIdParameterSet (預設) 
```
Remove-AzADUser -UPNOrObjectId <String> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### UPNParameterSet
```
Remove-AzADUser -UserPrincipalName <String> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

### ObjectIdParameterSet
```
Remove-AzADUser -ObjectId <String> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### DisplayNameParameterSet
```
Remove-AzADUser -DisplayName <String> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### InputObjectParameterSet
```
Remove-AzADUser -InputObject <PSADUser> [-PassThru] [-Force] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
刪除公司/ (帳戶的活動目錄使用者，也稱為組織識別碼) 。

## 例子

### 範例 1：根據使用者主體名稱移除使用者

```powershell
PS C:\> Remove-AzADUser -UserPrincipalName foo@domain.com
```

從租使用者移除具有使用者主體名稱 foo@domain.com "" 的使用者。

### 範例 2：根據物件識別碼移除使用者

```powershell
PS C:\> Remove-AzADUser -ObjectId 7a9582cf-88c4-4319-842b-7a5d60967a69
```

從租使用者移除物件識別碼為 '7a9582cf-88c4-4319-842b-7a5d60967a69'的使用者。

### 範例 3：使用管道移除使用者

```powershell
PS C:\> Get-AzADUser -ObjectId 7a9582cf-88c4-4319-842b-7a5d60967a69 | Remove-AzADUser
```

將使用者物件識別碼 '7a9582cf-88c4-4319-842b-7a5d60967a69' 和管道移至 Remove-AzADUser Cmdlet，將使用者從租使用者移除。
