# Get-AzADGroupMember

## 簡介
列出目前租使用者中 AD 群組的成員。

## 語法

### ObjectIdParameterSet (預設) 
```
Get-AzADGroupMember [-GroupObjectId <String>] [-DefaultProfile <IAzureContextContainer>] [-IncludeTotalCount]
 [-Skip <UInt64>] [-First <UInt64>] [<CommonParameters>]
```

### DisplayNameParameterSet
```
Get-AzADGroupMember -GroupDisplayName <String> [-DefaultProfile <IAzureContextContainer>] [-IncludeTotalCount]
 [-Skip <UInt64>] [-First <UInt64>] [<CommonParameters>]
```

### GroupObjectParameterSet
```
Get-AzADGroupMember -GroupObject <PSADGroup> [-DefaultProfile <IAzureContextContainer>] [-IncludeTotalCount]
 [-Skip <UInt64>] [-First <UInt64>] [<CommonParameters>]
```

## 描述
列出目前租使用者中 AD 群組的成員。

## 例子

### 範例 1：按 AD 群組物件識別碼列出成員

```powershell
PS C:\> Get-AzADGroupMember -GroupObjectId 85F89C90-780E-4AA6-9F4F-6F268D322EEE
```

以物件識別碼 '85F89C90-780E-4AA6-9F4F-6F268D322EEE'列出 AD 群組的成員。

### 範例 2：使用 paging 以 AD 群組物件識別碼列出成員

```powershell
PS C:\> Get-AzADGroupMember -GroupObjectId 85F89C90-780E-4AA6-9F4F-6F268D322EEE -First 100
```

以物件識別碼 '85F89C90-780E-4AA6-9F4F-6F268D322EEE'列出 AD 群組的前 100 個成員。

### 範例 3：以管道列出成員

```powershell
PS C:\> Get-AzADGroup -ObjectId 85F89C90-780E-4AA6-9F4F-6F268D322EEE | Get-AzADGroupMember
```

使用物件識別碼為 '85F89C90-780E-4AA6-9F4F-6F268D322EEE' 的 AD 群組，並管道到 Get-AzADGroupMember Cmdlet，以列出該群組中的所有成員。
