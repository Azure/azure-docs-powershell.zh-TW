# Get-AzLocation

## 簡介
獲得每個位置的所有位置和支援的資源提供者。

## 語法

```
Get-AzLocation [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 描述
**Get-AzLocation** Cmdlet 會取得每個位置的所有位置和支援的資源提供者。

## 例子

### 範例 1：取得所有位置和支援的資源提供者
```
PS C:\>Get-AzLocation
```

此命令會針對每個位置獲得所有位置和支援的資源提供者。
