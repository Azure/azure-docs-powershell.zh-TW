# Stop-AzDeployment

## 簡介
取消進行中部署

## 語法

### StopByDeploymentName (預設) 
```
Stop-AzDeployment [-Name] <String> [-PassThru] [-Pre] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### StopByDeploymentId
```
Stop-AzDeployment -Id <String> [-PassThru] [-Pre] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### StopByInputObject
```
Stop-AzDeployment -InputObject <PSDeployment> [-PassThru] [-Pre] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
**Stop-AzDeployment** Cmdlet 會取消訂閱範圍中已啟動但尚未完成的部署。
若要停止部署，部署必須擁有不完整的部署狀態，例如部署，而不是已完成的狀態，例如已部署或失敗。

若要在訂閱範圍中建立部署，請使用 New-AzDeployment Cmdlet。

此 Cmdlet 只會停止一個進行中的部署。 使用 *Name* 參數來停止特定部署。

## 例子

### 範例 1
```
PS C:\>Stop-AzDeployment -Name "deployment01"
```

此命令會取消目前訂閱範圍中執行中的部署「部署01」。

### 範例 2
```
PS C:\>Get-AzDeployment -Name "deployment01" | Stop-AzDeployment
```

此命令會獲得目前訂閱範圍中的部署「部署01」，並取消它。
