# Remove-AzTrafficManagerProfile

## 簡介
刪除流量管理員設定檔。

## 語法

### 領域
```
Remove-AzTrafficManagerProfile -Name <String> -ResourceGroupName <String> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### 物件
```
Remove-AzTrafficManagerProfile -TrafficManagerProfile <TrafficManagerProfile> [-Force]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 描述
**Remove-AzTrafficManagerProfile** Cmdlet 會刪除 Azure 流量管理員設定檔。
使用 Name 和 *ResourceGroupName* 參數指定要刪除的設定檔。 
或者，您可以使用 **TrafficManagerProfile 參數指定 TrafficManagerProfile** 物件，或者您可以使用管線。 

## 例子

### 範例 1：刪除由名稱指定的設定檔
```
PS C:\>Remove-AzTrafficManagerProfile -Name "ContosoProfile" -ResourceGroupName "ResourceGroup11"
```

此命令會刪除 ResourceGroup11 中名為 ContosoProfile 的設定檔。
命令會提示您確認。

### 範例 2：使用管線刪除設定檔
```
PS C:\>Get-AzTrafficManagerProfile -Name "ContosoProfile" -ResourceGroupName "ResourceGroup11" | Remove-AzTrafficManagerProfile -Force
```

此命令會獲得 ResourceGroup11 中名為 ContosoProfile 的設定檔。
命令接著使用管線運算子，將設定檔傳遞給 **Remove-AzTrafficManagerProfile** Cmdlet。
該 Cmdlet 會刪除該設定檔。
命令會指定 *Force* 參數。
因此，它不會提示您確認。
