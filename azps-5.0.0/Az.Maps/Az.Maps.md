# Az. 地圖模組
## 說明
本主題顯示 Azure Maps Cmdlet 的說明。

## Az. 地圖 Cmdlet
### [AzMapsAccount](Get-AzMapsAccount.md)
取得帳戶。

### [AzMapsAccountKey](Get-AzMapsAccountKey.md)
取得帳戶的 API 金鑰。
這些金鑰是在對 Azure 對應進行後續呼叫時所使用的驗證機制。

### [新-AzMapsAccount](New-AzMapsAccount.md)
建立 Azure 地圖帳戶。

### [新-AzMapsAccountKey](New-AzMapsAccountKey.md)
重新生成帳戶金鑰。

### [移除-AzMapsAccount](Remove-AzMapsAccount.md)
刪除 Azure 地圖帳戶。
