# Az ConnectedKubernetes 模組
## 說明
Microsoft Azure PowerShell： ConnectedKubernetes Cmdlet

## Az ConnectedKubernetes Cmdlet
### [AzConnectedKubernetes](Get-AzConnectedKubernetes.md)
傳回指定的連線群集的屬性，包括名稱、身分識別、屬性及其他群集詳細資料。

### [新-AzConnectedKubernetes](New-AzConnectedKubernetes.md)
使用 API 註冊新的 K8s 群集，進而在 ARM 中建立追蹤的資源

### [移除-AzConnectedKubernetes](Remove-AzConnectedKubernetes.md)
刪除連線的群集，並在 Azure 資源管理器中移除追蹤的資源 (ARM) 。

### [更新-AzConnectedKubernetes](Update-AzConnectedKubernetes.md)
用來更新已連接之群集資源之特定屬性的 API
