# Az CustomProviders 模組
## 說明
Microsoft Azure PowerShell：自訂提供者 Cmdlet

## Az CustomProviders Cmdlet
### [AzCustomProvider](Get-AzCustomProvider.md)
取得自訂資源提供者資訊清單。

### [AzCustomProviderAssociation](Get-AzCustomProviderAssociation.md)
取得關聯性。

### [新-AzCustomProvider](New-AzCustomProvider.md)
建立或更新自訂資源提供者。

### [新-AzCustomProviderAssociation](New-AzCustomProviderAssociation.md)
建立或更新關聯性。

### [移除-AzCustomProvider](Remove-AzCustomProvider.md)
刪除自訂資源提供者。

### [移除-AzCustomProviderAssociation](Remove-AzCustomProviderAssociation.md)
刪除關聯。

### [更新-AzCustomProvider](Update-AzCustomProvider.md)
更新現有的自訂資源提供者。
只能透過修補程式更新的值為標籤。
