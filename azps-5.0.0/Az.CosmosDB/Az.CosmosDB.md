# Az CosmosDB 模組
## 說明
本節中的主題會將 azure 資源管理器中的 azure PowerShell Cmdlet CosmosDB 到 Azure 資源管理員 (ARM) 架構中。 CosmosDB 命名空間中存在 Cmdlet。
