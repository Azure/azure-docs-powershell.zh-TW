# Remove-AzApplicationGatewayPrivateLinkConfiguration

## 摘要
從應用程式閘道移除 privateLink 設定。

## 句法

```
Remove-AzApplicationGatewayPrivateLinkConfiguration -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 說明
**AzApplicationGatewayPrivateLinkConfiguration** Cmdlet 會從 Azure 應用程式閘道中移除 privateLink 設定。

## 示例

### 範例1：移除應用程式閘道 PrivateLink 設定
```
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewayPrivateLinkConfiguration -ApplicationGateway $AppGw -Name "privateLinkConfig01"
```

第一個命令會取得應用程式閘道，並將它儲存在 $AppGw 變數中。
第二個命令會從 $AppGw 中儲存的應用程式閘道，移除名為 privateLinkConfig01 的 privateLink 設定。
