# Set-AzRouteTable

## 摘要
更新路由表。

## 句法

```
Set-AzRouteTable -RouteTable <PSRouteTable> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## 說明
**AzRouteTable** Cmdlet 會更新路由表。

## 示例

### 範例2：修改路由資料表

```
PS C:\> $rt = Get-AzRouteTable -ResourceGroupName "rgName" -Name "rtName"
PS C:\> $rt.DisableBgpRoutePropagation
False
PS C:\> $rt.DisableBgpRoutePropagation = $true
PS C:\> Set-AzRouteTable -RouteTable $rt
PS C:\> $rt = Get-AzRouteTable -ResourceGroupName "rgName" -Name "rtName"
PS C:\> $rt.DisableBgpRoutePropagation
True
```

第一個命令會取得名為 rtName 的路由資料表，並將它儲存在 $rt 變數中。
第二個命令會顯示 DisableBgpRoutePropagation 的值。
第三個命令更新 DisableBgpRoutePropagation 的值。
第四個命令會補救伺服器上的路由表。
第五個命令會取得更新的路由資料表，並將它儲存在 $rt 變數中。
第六個命令會顯示 DisableBgpRoutePropagation 的值。
