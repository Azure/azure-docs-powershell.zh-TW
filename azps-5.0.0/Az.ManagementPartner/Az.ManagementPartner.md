# Az ManagementPartner 模組
## 說明
Azure 資源管理員的管理合作夥伴 Cmdlet。

## Az ManagementPartner Cmdlet
### [AzManagementPartner](Get-AzManagementPartner.md)
取得 Microsoft 合作夥伴網路 (MPN 目前已驗證的使用者或服務主體) 識別碼。 

### [新-AzManagementPartner](New-AzManagementPartner.md)
將 Microsoft Partner Network (MPN) 識別碼與目前已驗證身份的使用者或服務主體相關聯。

### [移除-AzManagementPartner](Remove-AzManagementPartner.md)
刪除 [Microsoft 合作夥伴網路] (MPN 目前已驗證驗證的使用者或服務主體的) ID。

### [更新-AzManagementPartner](Update-AzManagementPartner.md)
更新 Microsoft 合作夥伴網路 (MPN 目前已驗證的使用者或服務主體) 識別碼。
