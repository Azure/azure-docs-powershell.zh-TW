# Get-AzsKeyVaultQuota

## 摘要
取得位置上 KeyVault 的所有配額物件的清單。

## 句法

```
Get-AzsKeyVaultQuota [[-Location] <String>] [<CommonParameters>]
```

## 說明
取得位置上 KeyVault 的所有配額物件的清單。

## 示例

### --------------------------範例 1--------------------------
```
Get-AzsKeyVaultQuota
```

取得位置上 KeyVault 的所有配額物件的清單。
