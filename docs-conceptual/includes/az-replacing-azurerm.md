> [!NOTE]
> 
> 在 2018 年 11 月之後，`AzureRM` 模組就不會新增任何功能或 Cmdlet。 `AzureRM` 會繼續提供支援及收到錯誤修正。 我們會在 `Az` 模組中提供新功能，同時達到 1.0。 `Az` 具有 `AzureRM` 的回溯相容性模式，其設計訴求是容易切換。 若要深入了解這個新模組以及如何升級，請參閱：
>
> * [Azure PowerShell Az 模組簡介](/powershell/azure/new-azureps-module-az)
> * [安裝 Azure PowerShell Az 模組](/powershell/azure/install-az-ps)
> * [遷移至新的 Azure PowerShell Az 模組](/powershell/azure/migrate-from-azurerm-to-az)
>
> 如果您的部署使用無法轉換的傳統部署模型，您可以安裝 Azure PowerShell 的服務管理版本。 如需詳細資訊，請參閱[安裝 Azure PowerShell 服務管理模組](/powershell/azure/servicemanagement/install-azure-ps)。
