# AzureRM 模組
## 說明
本節中的主題會將 azure 資源管理器中的 azure 保留的 Azure PowerShell Cmdlet 檔 (ARM) 架構中。 Cmdlet 會存在於 Microsoft Azure. 將命名空間保留在命令中。

## AzureRM Cmdlet
### [AzureRmReservation](Get-AzureRmReservation.md)
`Reservation`以指定的預留順序取得 s

### [AzureRmReservationCatalog](Get-AzureRmReservationCatalog.md)
取得可用保留的目錄。

### [AzureRmReservationHistory](Get-AzureRmReservationHistory.md)
取得的修訂歷程記錄 `Reservation` 。

### [AzureRmReservationOrder](Get-AzureRmReservationOrder.md)
取得 `ReservationOrder` 目前的租使用者。 如果提供了 [訂單識別碼]，則會檢索特定訂單。

### [AzureRmReservationOrderId](Get-AzureRmReservationOrderId.md)
取得 `ReservationOrder` 指定訂閱下的適用識別碼清單。

### [合併-AzureRmReservation](Merge-AzureRmReservation.md)
將兩個 `Reservation` s 合併成一個 `Reservation`

### [分割-AzureRmReservation](Split-AzureRmReservation.md)
將 a 分割 `Reservation` 成兩 `Reservation` 秒

### [更新-AzureRmReservation](Update-AzureRmReservation.md)
更新已套用的範圍 `Reservation` 。
