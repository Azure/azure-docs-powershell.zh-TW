# AzureRM Aks Module
## 說明
與 Azure 管理的 Kubernetes 群集互動的命令。

## AzureRM Aks Cmdlet
### [AzureRmAks](Get-AzureRmAks.md)
清單 Kubernetes 受管理的群集。

### [匯入-AzureRmAksCredential](Import-AzureRmAksCredential.md)
匯入及合併 managed Kubernetes 群集的 Kubectl config。

### [新-AzureRmAks](New-AzureRmAks.md)
建立新的 managed Kubernetes 群集。

### [移除-AzureRmAks](Remove-AzureRmAks.md)
刪除受管理的 Kubernetes 群集。

### [Set-AzureRmAks](Set-AzureRmAks.md)
更新或建立受管理的 Kubernetes 群集。

### [開始-AzureRmAksDashboard](Start-AzureRmAksDashboard.md)
建立 Kubectl SSH 隧道至 managed 群集的儀表板。

### [停止 AzureRmAksDashboard](Stop-AzureRmAksDashboard.md)
停止在 Start-AzureRmKubernetesDashboard 中建立的 Kubectl SSH 隧道。
