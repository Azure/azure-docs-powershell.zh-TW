# Set-AzureRmSqlServerRecommendedActionState

## 摘要
更新 Azure SQL Server 建議動作的狀態。

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 句法

```
Set-AzureRmSqlServerRecommendedActionState -RecommendedActionName <String> -State <RecommendedActionState>
 -ServerName <String> -AdvisorName <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 說明
**AzureRmSqlServerRecommendedActionState** Cmdlet 會更新 Azure SQL Server 建議動作的狀態。
這個 Cmdlet 會根據新的狀態來套用、還原或放棄建議的動作。
