# Remove-AzureRmNetworkInterface

## 摘要
移除網路介面。

[!INCLUDE [migrate-to-az-banner](../../includes/migrate-to-az-banner.md)]

## 句法

```
Remove-AzureRmNetworkInterface -Name <String> -ResourceGroupName <String> [-Force] [-PassThru] [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 說明
**AzureRmNetworkInterface** Cmdlet 會移除 Azure 網路介面。

## 示例

### 範例1：移除網路介面
```
PS C:\>Remove-AzureRmNetworkInterface -Name "NetworkInterface1" -ResourceGroup "ResourceGroup1"
```

這個命令會移除 [資源群組 ResourceGroup1] 中的 [網路介面 NetworkInterface1]。
由於不使用 *Force* 參數，系統會提示使用者確認此動作。

### 範例2：移除網路介面
```
PS C:\>Get-AzureRmNetworkInterface -ResourceGroupName "ResourceGroup1" | Remove-AzureRmNetworkInterface -Force
```

這個命令會移除 [資源群組 ResourceGroup1] 中的所有網路介面。
因為使用 *Force* 參數，系統不會提示使用者確認。
