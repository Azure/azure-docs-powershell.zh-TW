# Get-AzLoadBalancer

## 摘要
取得負載平衡器。

## 句法

### NoExpand
```
Get-AzLoadBalancer [-Name <String>] [-ResourceGroupName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

### 擴充
```
Get-AzLoadBalancer -Name <String> -ResourceGroupName <String> -ExpandResource <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 說明
**AzLoadBalancer** Cmdlet 會取得包含在資源群組中的一個或多個 Azure 負載平衡器。

## 示例

### 範例1：取得負載平衡器
```
PS C:\>Get-AzLoadBalancer -Name "MyLoadBalancer" -ResourceGroupName "MyResourceGroup"
```

這個命令會取得名為 MyLoadBalancer 的負載平衡器。
必須先存在負載平衡器，才能執行此 Cmdlet。
