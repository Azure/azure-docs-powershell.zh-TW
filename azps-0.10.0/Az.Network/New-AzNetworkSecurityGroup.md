# New-AzNetworkSecurityGroup

## 摘要
建立網路安全性群組。

## 句法

```
New-AzNetworkSecurityGroup -Name <String> -ResourceGroupName <String> -Location <String>
 [-SecurityRules <System.Collections.Generic.List`1[Microsoft.Azure.Commands.Network.Models.PSSecurityRule]>]
 [-Tag <Hashtable>] [-Force] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 說明
**新的-AzNetworkSecurityGroup** Cmdlet 會建立 Azure 網路安全性群組。

## 示例

### 1：建立新的 [網路 securtiy] 群組
```
New-AzNetworkSecurityGroup -Name "nsg1" -ResourceGroupName "rg1"  -Location  "westus"
```

這個命令會 ceates 「westus」資源群組「rg1」中名為「nsg1」的新 Azure 網路安全性群組。

### 2：建立詳細的網路安全群組
```
$rule1 = New-AzNetworkSecurityRuleConfig -Name rdp-rule -Description "Allow RDP"
    -Access Allow -Protocol Tcp -Direction Inbound -Priority 100 -SourceAddressPrefix
    Internet -SourcePortRange * -DestinationAddressPrefix * -DestinationPortRange 3389

$rule2 = New-AzNetworkSecurityRuleConfig -Name web-rule -Description "Allow HTTP"
    -Access Allow -Protocol Tcp -Direction Inbound -Priority 101 -SourceAddressPrefix
    Internet -SourcePortRange * -DestinationAddressPrefix * -DestinationPortRange 80

$nsg = New-AzNetworkSecurityGroup -ResourceGroupName TestRG -Location westus -Name
    "NSG-FrontEnd" -SecurityRules $rule1,$rule2
```

步驟：1建立允許從網際網路存取到埠3389的安全規則。
步驟：2建立允許從網際網路存取到埠80的安全規則。
步驟：3將上述建立的規則新增至名為 NSG-前端的新 NSG。
