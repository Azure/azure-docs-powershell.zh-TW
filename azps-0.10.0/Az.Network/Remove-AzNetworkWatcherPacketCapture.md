# Remove-AzNetworkWatcherPacketCapture

## 摘要
移除 [資料包捕獲資源]。

## 句法

### SetByResource (預設) 
```
Remove-AzNetworkWatcherPacketCapture -NetworkWatcher <PSNetworkWatcher> -PacketCaptureName <String>
 [-PassThru] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### SetByName
```
Remove-AzNetworkWatcherPacketCapture -NetworkWatcherName <String> -ResourceGroupName <String>
 -PacketCaptureName <String> [-PassThru] [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## 說明
Remove-AzNetworkWatcherPacketCapture 會移除 [資料包捕獲資源]。 建議您先呼叫 Stop-AzNetworkWatcherPacketCapture，然後再呼叫 AzNetworkWatcherPacketCapture。 如果 Remove-AzNetworkWatcherPacketCapture 稱為 [資料包捕獲會話]，則可能無法儲存 [資料包捕獲]。 如果在移除前停止會話，則不會移除內含捕獲資料的 .cap 檔案。 

## 示例

### ---範例1：移除資料包捕獲會話--
```
Remove-AzNetworkWatcherPacketCapture -NetworkWatcher $networkWatcher -PacketCaptureName "PacketCaptureTest"
```

在這個範例中，我們會移除名為 "PacketCaptureTest" 的現有資料包捕獲會話。
