# Start-AzVM

## 摘要
啟動 Azure 虛擬機器。

## 句法

### ResourceGroupNameParameterSetName (預設) 
```
Start-AzVM [-Name] <String> [-ResourceGroupName] <String> [-AsJob]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

### IdParameterSetName
```
Start-AzVM [-Name] <String> [-Id] <String> [-AsJob] [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## 說明
**AzVM** Cmdlet 會啟動 Azure 虛擬機器。

## 示例

### 範例1：啟動虛擬機器
```
PS C:\> Start-AzVM -ResourceGroupName "ResourceGroup11" -Name "VirtualMachine07"
```

這個命令會在 ResourceGroup11 中啟動名為 VirtualMachine07 的虛擬機器。
