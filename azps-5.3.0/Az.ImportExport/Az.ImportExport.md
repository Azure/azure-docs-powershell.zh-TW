# Az ImportExport 模組
## 說明
Microsoft Azure PowerShell： ImportExport Cmdlet

## Az ImportExport Cmdlet
### [AzImportExport](Get-AzImportExport.md)
取得現有作業的相關資訊。

### [AzImportExportBitLockerKey](Get-AzImportExportBitLockerKey.md)
傳回指定作業中所有磁片磁碟機的 BitLocker 金鑰。

### [AzImportExportLocation](Get-AzImportExportLocation.md)
傳回與匯入或匯出作業相關聯之磁片之位置的詳細資料。
位置是 Azure 區域。

### [新-AzImportExport](New-AzImportExport.md)
在指定的訂閱中建立新作業或更新現有作業。

### [新-AzImportExportDriveListObject](New-AzImportExportDriveListObject.md)
建立 ImportExport 的 DriverList 物件。

### [移除-AzImportExport](Remove-AzImportExport.md)
刪除現有的工作。
只有 [建立] 或 [已完成] 狀態中的工作可以刪除。

### [更新-AzImportExport](Update-AzImportExport.md)
更新作業的特定屬性。
您可以呼叫此操作，通知匯入/匯出服務，包含匯入或匯出作業的硬驅已傳送至 Microsoft 資料中心。
您也可以用來取消現有作業。
