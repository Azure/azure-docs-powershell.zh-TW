# Remove-AzApplicationGatewaySslProfile

## 摘要
從應用程式閘道移除 ssl 設定檔。

## 句法

```
Remove-AzApplicationGatewaySslProfile -Name <String> -ApplicationGateway <PSApplicationGateway>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 說明
**AzApplicationGatewaySslProfile** Cmdlet 會從應用程式閘道中移除 ssl 設定檔。

## 示例

### 範例1
```powershell
PS C:\>$AppGw = Get-AzApplicationGateway -Name "ApplicationGateway01" -ResourceGroupName "ResourceGroup01"
PS C:\> Remove-AzApplicationGatewaySslProfile -ApplicationGateway $AppGw -Name "SslProfile01"
```

第一個命令會取得一個名為 ApplicationGateway01 的應用程式閘道，該閘道屬於名為 ResourceGroup01 的資源群組，並將它儲存在 $AppGw 變數中。 第二個命令會從儲存在 $AppGw 的應用程式閘道中，移除名為 SslProfile01 的 ssl 設定檔。
