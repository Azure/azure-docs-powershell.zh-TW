# Get-AzNetworkServiceTag

## 摘要
取得服務標記資訊資源的清單。

## 句法

```
Get-AzNetworkServiceTag -Location <String> [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 說明
**AzNetworkServiceTag** Cmdlet 會取得服務標記資訊資源的清單。

請注意，您指定的 Azure 區域資訊將用來做為版本 (的參照，而不是根據位置) 的篩選。 例如，即使您指定的是在 `-Location eastus2` 所有地區都要有首碼詳細資料的服務標記清單，但僅限您訂閱所屬的雲端 (（例如，上市公司、美國政府、中國或德國）) 。
