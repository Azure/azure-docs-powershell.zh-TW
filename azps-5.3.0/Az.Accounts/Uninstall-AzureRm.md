# Uninstall-AzureRm

## 摘要
從電腦中移除所有 AzureRm 模組。

## 句法

```
Uninstall-AzureRm [-PassThru] [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 說明
從電腦中移除所有 AzureRm 模組。

## 示例

### 範例1
```
PS C:\> Uninstall-AzureRm
```

執行此命令時，會從電腦的所有 AzureRm 模組中移除執行 Cmdlet 的 PowerShell 版本。
