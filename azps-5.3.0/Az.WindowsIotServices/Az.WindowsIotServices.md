# Az WindowsIotServices 模組
## 說明
Microsoft Azure PowerShell： WindowsIotServices Cmdlet

## Az WindowsIotServices Cmdlet
### [AzWindowsIotServicesDevice](Get-AzWindowsIotServicesDevice.md)
取得 Windows IoT 裝置服務的非安全性相關中繼資料。

### [新-AzWindowsIotServicesDevice](New-AzWindowsIotServicesDevice.md)
建立或更新 Windows IoT 裝置服務的中繼資料。
修改屬性的一般模式是檢索 Windows IoT 裝置服務中繼資料與安全性中繼資料，然後將它們與新主體中修改過的值合併，以更新 Windows IoT 裝置服務。

### [移除-AzWindowsIotServicesDevice](Remove-AzWindowsIotServicesDevice.md)
刪除 Windows IoT 裝置服務。

### [更新-AzWindowsIotServicesDevice](Update-AzWindowsIotServicesDevice.md)
更新 Windows IoT 裝置服務的中繼資料。
修改屬性的一般模式是檢索 Windows IoT 裝置服務中繼資料與安全性中繼資料，然後將它們與新主體中修改過的值合併，以更新 Windows IoT 裝置服務。
