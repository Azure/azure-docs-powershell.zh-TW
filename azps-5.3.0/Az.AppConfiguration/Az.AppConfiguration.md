# Az AppConfiguration 模組
## 說明
Microsoft Azure PowerShell： AppConfiguration Cmdlet

## Az AppConfiguration Cmdlet
### [AzAppConfigurationStore](Get-AzAppConfigurationStore.md)
取得或列出 app 設定存放區。

### [AzAppConfigurationStoreKey](Get-AzAppConfigurationStoreKey.md)
列出指定的配置存放區的便捷鍵。

### [新-AzAppConfigurationStore](New-AzAppConfigurationStore.md)
使用指定的參數建立配置存放區。

### [新-AzAppConfigurationStoreKey](New-AzAppConfigurationStoreKey.md)
針對指定的配置存放區重新產生便捷鍵。

### [移除-AzAppConfigurationStore](Remove-AzAppConfigurationStore.md)
刪除配置存放區。

### [Test-AzAppConfigurationStoreNameAvailability](Test-AzAppConfigurationStoreNameAvailability.md)
檢查 [配置] 儲存區名稱是否可供使用。

### [更新-AzAppConfigurationStore](Update-AzAppConfigurationStore.md)
使用指定的參數更新配置存儲區。
