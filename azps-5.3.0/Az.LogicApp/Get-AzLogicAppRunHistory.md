# Get-AzLogicAppRunHistory

## 摘要
取得邏輯 app 的執行歷程記錄。

## 句法

```
Get-AzLogicAppRunHistory -ResourceGroupName <String> -Name <String> [-RunName <String>] [-FollowNextPageLink]
 [-MaximumFollowNextPageLink <Int32>] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 說明
**AzLogicAppRunHistory** Cmdlet 會取得邏輯 app 的執行歷程記錄。
這個 Cmdlet 會傳回 **WorkflowRun** 物件的集合。
指定邏輯 app 與資源群組。
此模組支援動態參數。
若要使用動態參數，請在命令中輸入。
若要探索動態參數的名稱，請在 Cmdlet 名稱後輸入連字號 ( ) ，然後重複按 Tab 鍵以迴圈顯示可用的參數。
如果您省略必要的範本參數，則 Cmdlet 會提示您輸入值。

這個命令會取得名為 LogicApp03 的邏輯 app 的執行歷程記錄。

### 範例2：取得邏輯 app 執行
```powershell
PS C:\>Get-AzLogicAppRunHistory -ResourceGroupName "Resourcegroup11" -Name "LogicApp03" -RunName "08587489104702792076"
CorrelationId    : 55830326-9042-404d-a4c3-fab198106a57
EndTime          : 1/13/2016 2:46:55 PM
Error            : {code, message}
Name             : 08587489104702792076
Outputs          : {}
StartTime        : 1/13/2016 2:46:55 PM
Status           : Failed
TriggerName      : 
LogicAppName     : LogicApp03
LogicAppVersion  : 08587489107859952120
```

這個命令會針對名為 LogicApp03 的邏輯 app 取得特定的邏輯 app 執行。

### 範例3

```powershell
Get-AzLogicAppRunHistory -Name 'LogicApp03' -ResourceGroupName MyResourceGroup -FollowNextPageLink
```

這個命令會透過遵循 NextPageLink 來取得名為 LogicApp03 之邏輯 app 的整個執行歷程記錄。

### 範例4

```powershell
Get-AzLogicAppRunHistory -Name 'LogicApp03' -ResourceGroupName MyResourceGroup -FollowNextPageLink -MaximumFollowNextPageLink 1
```

這個命令會透過遵循 NextPageLink 並將結果大小限制為兩個頁面，以取得名為 LogicApp03 之邏輯 app 的前兩頁。
每個頁面都包含三十個結果。
