# Get-AzPrivateDnsVirtualNetworkLink

## 摘要
取得與指定的專用 DNS 區域相關聯的虛擬網路連結。

## 句法

```
Get-AzPrivateDnsVirtualNetworkLink -ResourceGroupName <String> -ZoneName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 說明
**AzPrivateDnsVirtualNetworkLink** Cmdlet 會從指定的資源群組取得與特定專用 DNS 區域相關聯的虛擬網路連結。
如果您指定 *Name* 參數，則會傳回單一 **PSPrivateDnsVirtualNetworkLink** 物件。
如果您沒有指定 *Name* 參數，則會傳回包含與指定資源群組中之區域相關聯之所有連結的陣列。
您可以使用 **PSPrivateDnsVirtualNetworkLink** 物件來更新連結。

```
PS C:\> $Link = Get-AzPrivateDnsVirtualNetworkLink -ResourceGroupName "MyResourceGroup" -ZoneName "myzone.com" -Name "mylink"

ZoneName                : myzone.com

                          etwork/virtualNetworks/myvirtualnetwork
Location                :
Etag                    : "xxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx"
Tags                    : {tag1}
RegistrationEnabled     : True
VirtualNetworkLinkState : Completed
ProvisioningState       : Succeeded
```
