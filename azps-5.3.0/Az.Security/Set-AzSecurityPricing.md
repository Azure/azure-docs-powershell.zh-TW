# Set-AzSecurityPricing

## 摘要

啟用或停用 Azure 安全中心訂閱的 Azure Defender 方案。

## 句法

### SubscriptionLevelResource (預設) 

```powershell
Set-AzSecurityPricing -Name <String> -PricingTier <String> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

### InputObject

```powershell
Set-AzSecurityPricing -InputObject <PSSecurityPricing> [-DefaultProfile <IAzureContextContainer>] [-WhatIf]
 [-Confirm] [<CommonParameters>]
```

## 說明

啟用或停用任何訂閱的 Azure Defender 方案。

如需 Azure Defender 和可用方案的詳細資訊，請參閱 [Azure Defender 簡介](https://docs.microsoft.com/azure/security-center/azure-defender)。

## 示例

### 範例1

```powershell
PS C:\> Set-AzSecurityPricing -Name "virtualmachines" -PricingTier "Standard"
```

為 **伺服器啟用 Azure Defender** for 訂閱。

"標準" 指的是 azure Defender 方案的「開啟」狀態，如 azure 安全中心的 azure 入口網站的 [定價] 和 [設定] 區域所示。
