# Get-AzSqlDatabase

## 摘要
取得一或多個資料庫。

## 句法

```
Get-AzSqlDatabase [[-DatabaseName] <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 說明
**AzSqlDatabase** Cmdlet 會從 Azure Sql Database Server 取得一個或多個 azure sql 資料庫。
Azure 上的 SQL Server Stretch Database 服務也支援此 Cmdlet。
