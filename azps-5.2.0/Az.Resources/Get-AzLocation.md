# Get-AzLocation

## 摘要
取得每個位置的所有位置和支援的資源提供者。

## 句法

```
Get-AzLocation [-ApiVersion <String>] [-Pre] [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 說明
**AzLocation** Cmdlet 會取得所有位置，以及每個位置支援的資源提供者。

## 示例

### 範例1：取得所有位置及支援的資源提供者
```
PS C:\>Get-AzLocation
```

這個命令會取得所有位置，以及每個位置支援的資源提供者。
