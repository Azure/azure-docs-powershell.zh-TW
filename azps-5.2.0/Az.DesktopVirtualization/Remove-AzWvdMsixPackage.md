# Remove-AzWvdMsixPackage

## 摘要
移除 MSIX 套件。

## 句法

### 刪除 (預設) 
```
Remove-AzWvdMsixPackage -FullName <String> -HostPoolName <String> -ResourceGroupName <String>
 [-SubscriptionId <String>] [-DefaultProfile <PSObject>] [-PassThru] [-Confirm] [-WhatIf] [<CommonParameters>]
```

### DeleteViaIdentity
```
Remove-AzWvdMsixPackage -InputObject <IDesktopVirtualizationIdentity> [-DefaultProfile <PSObject>] [-PassThru]
 [-Confirm] [-WhatIf] [<CommonParameters>]
```

## 說明
移除 MSIX 套件。

## 示例

### 範例1：依套件完整名稱刪除 MSIX 套件
```powershell
PS C:\> Remove-AzWvdMsixPackage -HostPoolName HostPoolName -ResourceGroupName ResourceGroupName -SubscriptionId SubscriptionId -FullName PackageFullName
```

這個命令會刪除 HostPool 中的 MSIX 套件

別名

複雜的參數屬性

若要建立如下所述的參數，請構造包含適當屬性的雜湊資料表。 如需雜湊資料表的詳細資訊，請執行 Get-Help about_Hash_Tables。


INPUTOBJECT <IDesktopVirtualizationIdentity> ：身分識別參數
  - `[ApplicationGroupName <String>]`：應用程式群組的名稱
  - `[ApplicationName <String>]`：指定的應用程式群組中的應用程式名稱
  - `[DesktopName <String>]`：指定的桌面群組中的桌面名稱
  - `[HostPoolName <String>]`：指定資源群組中的主機池名稱
  - `[Id <String>]`：資源身分識別路徑
  - `[MsixPackageFullName <String>]`： MSIX 套件在指定 hostpool 中的版本特定套件完整名稱
  - `[ResourceGroupName <String>]`：資源群組的名稱。 名稱不區分大小寫。
  - `[SessionHostName <String>]`：位於指定主機池中的工作階段主機名稱
  - `[SubscriptionId <String>]`：目標訂閱的識別碼。
  - `[UserSessionId <String>]`：指定工作階段主機中的使用者會話名稱
  - `[WorkspaceName <String>]`：工作區的名稱
