# Az. 安全性模組
## 說明
Azure 安全性中心可讓您控制 Azure 訂閱的安全性，以及您在 Azure 以外連線至該電腦的其他電腦。

## Az. 安全性 Cmdlet
### [Disable-AzIotSecurityAnalyticsAggregatedAlert](Disable-AzIotSecurityAnalyticsAggregatedAlert.md)
關閉 Iot 匯總通知
