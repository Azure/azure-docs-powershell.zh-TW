# Save-AzureWebsiteLog

## 摘要
下載並儲存指定網站的記錄。

## 句法

```
Save-AzureWebsiteLog [-Output <String>] [-PassThru] [-Name <String>] [-Slot <String>]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## 說明
本主題描述 Microsoft Azure PowerShell 模組的0.8.10 版本中的 Cmdlet。
若要取得您正在使用的模組版本，請在 Azure PowerShell 主控台輸入 `(Get-Module -Name Azure).Version` 。

**AzureWebsiteLog** Cmdlet 會下載指定網站的記錄。

## 示例

### 範例1：下載並儲存網站的記錄
```
PS C:\> Save-AzureWebsiteLogs -Name mySite -Output .\logs.zip
```

這個範例會將網站網站的執行時間和部署記錄下載至目前目錄中 logs.zip 的檔案。
