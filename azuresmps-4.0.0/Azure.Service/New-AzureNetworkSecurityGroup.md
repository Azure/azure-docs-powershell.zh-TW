# New-AzureNetworkSecurityGroup

## 摘要
建立 Azure 網路安全性群組。

## 句法

```
New-AzureNetworkSecurityGroup -Name <String> -Location <String> [-Label <String>] [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## 說明
**新的-AzureNetworkSecurityGroup** Cmdlet 會在位置建立 Azure 網路安全性群組。
