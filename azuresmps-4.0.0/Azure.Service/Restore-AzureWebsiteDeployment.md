# Restore-AzureWebsiteDeployment

## 摘要
在 Azure 中 Redeploys 網站的先前部署。

## 句法

```
Restore-AzureWebsiteDeployment [-CommitId <String>] [-Force] [-Name <String>] [-Slot <String>]
 [-Profile <AzureSMProfile>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 說明
本主題描述 Microsoft Azure PowerShell 模組的0.8.10 版本中的 Cmdlet。
若要取得您正在使用的模組版本，請在 Azure PowerShell 主控台輸入 `(Get-Module -Name Azure).Version` 。

**Restore-AzureWebsiteDeployment** Cmdlet 會 redeploys Azure 中先前的網站部署。
這個程式會將目前的部署取代為選取的部署。

## 示例

### 範例1：重新部署網站
```
PS C:\> Restore-AzureWebsiteDeployment -Name "ContosoSite" -CommitId "f876543210"
```

這個命令 redeploys 具有名為 ContosoSite 之網站之 ID f876543210 的部署。
