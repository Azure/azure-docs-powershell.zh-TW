# Disable-AzureServiceProjectRemoteDesktop

## 摘要
停用遠端桌面電腦對雲端服務的存取權。

## 句法

```
Disable-AzureServiceProjectRemoteDesktop [-PassThru] [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## 說明
本主題描述 Microsoft Azure PowerShell 模組的0.8.10 版本中的 Cmdlet。
若要取得您正在使用的模組版本，請在 Azure PowerShell 主控台輸入 `(Get-Module -Name Azure).Version` 。

**Disable-AzureServiceProjectRemoteDesktop** 會停用遠端桌面對託管服務的存取權。
在停用遠端桌面存取之後，您必須使用 **Publish AzureServiceProject** Cmdlet 發佈服務，變更才會生效。
