# Get-AzureRemoteAppLocation

## 摘要
列出 Azure RemoteApp 支援的位置。

## 句法

```
Get-AzureRemoteAppLocation [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## 說明
**AzureRemoteAppLocation** Cmdlet 會列出 Azure RemoteApp 支援的位置。
