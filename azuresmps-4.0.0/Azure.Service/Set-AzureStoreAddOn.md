# Set-AzureStoreAddOn

## 摘要
更新現有的附加元件實例。

## 句法

```
Set-AzureStoreAddOn -Name <String> -Plan <String> [-PromotionCode <String>] [-PassThru]
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## 說明
本主題描述 Microsoft Azure PowerShell 模組的0.8.10 版本中的 Cmdlet。
若要取得您正在使用的模組版本，請在 Azure PowerShell 主控台輸入 `(Get-Module -Name Azure).Version` 。

這個 Cmdlet 會從目前的訂閱更新現有的附加元件實例。

## 示例

### 範例1
```
PS C:\> Set-AzureStoreAddOn MyAddOn NewPlanId
```

這個範例會使用新的方案識別碼來更新附加元件。

### 範例2
```
PS C:\> Set-AzureStoreAddOn MyAddOn NewPlanId MyPromoCode
```

這個範例會使用新的方案識別碼及促銷代碼更新附加元件。
