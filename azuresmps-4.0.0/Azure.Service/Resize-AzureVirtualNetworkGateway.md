# Resize-AzureVirtualNetworkGateway

## 摘要
重新調整虛擬閘道的大小。

## 句法

```
Resize-AzureVirtualNetworkGateway -GatewayId <String> -GatewaySKU <String> [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## 說明
Resize-AzureVirtualNetworkGateway Cmdlet 會調整虛擬網路閘道的大小。
