# Remove-AzureMediaServicesAccount

## 摘要
移除指定的 Azure 媒體服務帳戶。

**注意：**  此版本已棄用，請參閱 [較新的版本](https://docs.microsoft.com/powershell/module/azurerm.media/?view=azurermps-5.4.0#media_services)。

## 句法

```
Remove-AzureMediaServicesAccount -Name <String> [-Force] [-Profile <AzureSMProfile>] [-WhatIf] [-Confirm]
 [<CommonParameters>]
```

## 說明
本主題描述 Microsoft Azure PowerShell 模組的0.8.10 版本中的 Cmdlet。
若要取得您正在使用的模組版本，請在 Azure PowerShell 主控台輸入 `(Get-Module -Name Azure).Version` 。

**AzureMediaServicesAccount** Cmdlet 會移除媒體服務帳戶。

## 示例

### 範例1：刪除媒體服務帳戶
```
PS C:\> Remove-AzureMediaServicesAccount -Name "mediaservicesaccount" -Force
```
