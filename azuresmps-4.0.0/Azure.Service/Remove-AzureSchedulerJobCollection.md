# Remove-AzureSchedulerJobCollection

## 摘要
刪除排程作業集合。

## 句法

```
Remove-AzureSchedulerJobCollection [-Force] [-Location <String>] -JobCollectionName <String>
 [-Profile <AzureSMProfile>] [<CommonParameters>]
```

## 說明
本主題描述 Microsoft Azure PowerShell 模組的0.8.10 版本中的 Cmdlet。
若要取得您正在使用的模組版本，請在 Azure PowerShell 主控台輸入 `(Get-Module -Name Azure).Version` 。

**AzureSchedulerJobCollection** Cmdlet 會刪除排程作業集合以及該集合下的任何作業。

## 示例

### 範例1：刪除某個位置的作業集合
```
PS C:\> Remove-AzureSchedulerJobCollection -Location "North Central US" -JobCollectionName "JobCollection01"
```

這個命令會在北中美位置刪除名為 JobCollection01 的排程作業集合。
此命令也會刪除 [JobCollection01] 下的工作。

### 範例2：刪除作業位置
```
PS C:\> Remove-AzureSchedulerJobCollection -JobCollectionName "JobCollection02"
```

這個命令會刪除名為 JobCollection02 的排程作業集合。
此命令也會刪除 [JobCollection02] 下的工作。
