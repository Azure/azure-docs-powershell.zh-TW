# Get-AzureRemoteAppSession

## 摘要
列出集合的所有活動和中斷連接的 Azure RemoteApp 會話。

## 句法

```
Get-AzureRemoteAppSession [-CollectionName] <String> [[-UserUpn] <String>] [-Profile <AzureSMProfile>]
 [<CommonParameters>]
```

## 說明
**AzureRemoteAppSession** Cmdlet 會列出 azure remoteapp 集合的所有活動與中斷連線的 azure RemoteApp 會話。

## 示例

### 範例1：列出集合中的所有會話
```
PS C:\> Get-AzureRemoteAppSession -CollectionName "ContosoApps"
```

這個命令會列出 Azure RemoteApp 集合中名為 ContosoApps 的所有會話。
