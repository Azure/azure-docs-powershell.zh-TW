# Azs. 系統管理模組
## 說明
預覽版本的 AzureStack backup 接線員 module （預覽版）。  這個單元可讓操作員監視及管理備份。

## Azs Backup. 系統管理 Cmdlet
### [AzsBackup](Get-AzsBackup.md)
從以名稱為基礎的位置傳回備份。

### [AzsBackupConfiguration](Get-AzsBackupConfiguration.md)
傳回備份配置清單。

### [新-AzsEncryptionKeyBase64](New-AzsEncryptionKeyBase64.md)
產生基礎結構備份的加密金鑰。

### [Restore-AzsBackup](Restore-AzsBackup.md)
還原備份。

### [Set-AzsBackupConfiguration](Set-AzsBackupConfiguration.md)
在指定的位置設定備份配置。

### [開始-AzsBackup](Start-AzsBackup.md)
備份特定位置。
