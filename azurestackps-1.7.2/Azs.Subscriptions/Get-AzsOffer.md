# Get-AzsOffer

## 摘要
取得公開優惠清單。

## 句法

```
Get-AzsOffer [[-Skip] <Int32>] [[-Top] <Int32>] [<CommonParameters>]
```

## 說明
取得公開優惠清單。

## 示例

### --------------------------範例 1--------------------------
```
Get-AzsOffer | fl
```

取得公開優惠清單。
