# Get-AzsSubscription

## 摘要
取得訂閱清單。

## 句法

###  (預設) 清單
```
Get-AzsSubscription [<CommonParameters>]
```

### 獲取
```
Get-AzsSubscription [-SubscriptionId] <String> [<CommonParameters>]
```

## 說明
取得訂閱清單。

## 示例

### 範例1
```
Get-AzsSubscription
```

取得訂閱清單。
