# Set-AzFrontDoor

## 摘要
更新前門負載平衡器

## 句法

```
 [-BackendPool <PSBackendPool[]>] [-FrontendEndpoint <PSFrontendEndpoint[]>]
 [-LoadBalancingSetting <PSLoadBalancingSetting[]>] [-HealthProbeSetting <PSHealthProbeSetting[]>]
```

## 說明
**AzFrontDoor** Cmdlet 會更新前門負載平衡器。 如果未提供輸入參數，則會使用現有的前蓋的舊參數。

## 示例

