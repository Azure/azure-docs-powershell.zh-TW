# Get-AzIntegrationAccountCertificate

## 摘要
從資源群組取得整合帳戶憑證。

## 句法

```
Get-AzIntegrationAccountCertificate [-ResourceGroupName <String>] [-Name <String>] [-CertificateName <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 說明
**AzIntegrationAccountCertificate** Cmdlet 會從資源群組取得整合帳戶憑證。
指定整合帳戶名稱、資源群組名稱及憑證名稱。
此模組支援動態參數。
若要使用動態參數，請在命令中輸入。
若要探索動態參數的名稱，請在 Cmdlet 名稱後輸入連字號 ( ) ，然後重複按 Tab 鍵以迴圈顯示可用的參數。
如果您省略必要的範本參數，則 Cmdlet 會提示您輸入值。
