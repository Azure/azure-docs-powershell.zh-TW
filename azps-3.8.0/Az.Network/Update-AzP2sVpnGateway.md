# Update-AzP2sVpnGateway

## 摘要
在 [VirtualHub] 底下的 [網站連線] 底下，更新現有的 P2SVpnGateway。

## 句法

### ByP2SVpnGatewayNameNoVpnServerConfigurationUpdate (預設) 
```
Update-AzP2sVpnGateway -ResourceGroupName <String> -Name <String> [-VpnClientAddressPool <String[]>]
 [-VpnGatewayScaleUnit <UInt32>] [-Tag <Hashtable>] [-AsJob] [-DefaultProfile <IAzureContextContainer>]
 [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 說明
**AzP2sVpnGateway** Cmdlet 可讓您在 VirtualHub 中使用新的 VpnClientAddressPool 或新 VpnServerConfiguration 或變更 VpnGatewayScaleUnit 來更新現有 P2SVpnGateway。
