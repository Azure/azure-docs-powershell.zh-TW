# Get-AzCognitiveServicesAccountNetworkRuleSet

## 摘要
取得認知服務帳戶的 NetworkRule 屬性

## 句法

```
Get-AzCognitiveServicesAccountNetworkRuleSet [-ResourceGroupName] <String> [-Name] <String>
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 說明
**AzCognitiveServicesAccountNetworkRuleSet** Cmdlet 會取得認知服務帳戶的 NetworkRule 屬性

## 示例

### 範例1：取得指定的認知服務帳戶的 NetworkRule 屬性
```
PS C:\> Get-AzCognitiveServicesAccountNetworkRuleSet  -ResourceGroupName "rg1" -Name "myaccount"
```

這個命令會取得指定的認知服務帳戶的 NetworkRule 屬性
