# Get-AzAvailabilitySet

## 摘要
取得資源群組中的 Azure 可用性集合。

## 句法

```
Get-AzAvailabilitySet [[-ResourceGroupName] <String>] [[-Name] <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## 說明
**AzAvailabilitySet** Cmdlet 可在資源群組中取得 Azure 可用性集。
您可以指定要取得的特定可用性集的名稱。

## 示例
