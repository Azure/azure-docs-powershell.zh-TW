# Get-AzSqlDatabaseReplicationLink

## 摘要
取得 Azure SQL 資料庫與資源群組或 SQL Server 之間的異地複製連結。

## 句法

```
Get-AzSqlDatabaseReplicationLink [-DatabaseName] <String> -PartnerResourceGroupName <String>
 [-PartnerServerName <String>] [-ServerName] <String> [-ResourceGroupName] <String>
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 說明
**AzSqlDatabaseReplicationLink** Cmdlet 會取代 **AzSqlDatabaseCopy** Cmdlet。
它會取得指定 Azure SQL 資料庫與資源群組或 AzureSQL 伺服器之間的所有異地複製連結。
