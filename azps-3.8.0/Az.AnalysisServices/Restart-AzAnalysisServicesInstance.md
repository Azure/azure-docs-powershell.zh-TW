# Restart-AzAnalysisServicesInstance

## 摘要
在目前登入的環境中重新開機 Analysis Services 伺服器的實例，如 Add-AzAnalysisServicesAccount 命令中所指定

## 句法

```
Restart-AzAnalysisServicesInstance [-Instance] <String> [-PassThru] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## 說明
Restart-AzAnalysisServicesInstance Cmdlet 會重新開機 Azure Analysis Services 伺服器的實例

## 示例

### 範例1
```
PS C:\>Restart-AzAnalysisServicesInstance
Instance: testserver
```

這個命令會在 Add-AzAnalysisServicesAccount 命令中指定的環境中，重新開機伺服器「testserver」
